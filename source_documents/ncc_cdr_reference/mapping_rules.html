EDR mapping rules

This section lists the rules for populating the fields of different output records, where "output record" and "output field" refer to the record types and tags in the predefined ASN.1 schema. Similarly "input field" refers to the fields/tags used in the corresponding EDRs in the NCC generic CDR schema.

The fields for the top level fields and blocks are given in an order corresponding to the tag numbers for the fields in the predefined ASN.1 schema. Fields for different EDR types describes which fields are included in which output EDRs. The rules for the fields under the top level are in the order in which they appear in the output.

A rule for generating an output field does not depend on the output record type, unless differently stated.
