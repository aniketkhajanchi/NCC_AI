In Terraform

Compute:

Create a nodepool that has a taint with name compute

taints = [{
  key = "compute"
  value = "true"
  effect = "NO_SCHEDULE"
}]

Set a label also with the name compute

labels = {
  compute = "true"
}

Aerospike:

Create a nodepool that has a taint with name aerospike

taints = [{
  key = "aerospike"
  value = "true"
  effect = "NO_SCHEDULE"
}]

Set a label also with the name compute

labels = {
  aerospike = "true"
}

General purpose

This is the default. Create a third node pool that does not have taints or labels.
