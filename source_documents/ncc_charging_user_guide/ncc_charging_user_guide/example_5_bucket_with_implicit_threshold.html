Example 5: Bucket with implicit threshold

During slice reduction if no upcoming threshold is present for bucket then the implicit threshold of 100% is used to compute the delta. This example explains this scenario:

Preconditions:

  1. Quota management (slicing profile): min slice size = 5 B, slice size = 1024, and Slice Allocation Factor (SAF) = 10%.
  2. There is Cap at 50 B with Send-RAR action. Cap is applied on the counter.
  3. There is a Bucket having initial value of 5000 B.

Execution:

  1. Execute a SCUR call.
  2. For CCR-I, execute a centralized call in which granted units = 5 B.
  3. For CCR-U, execute a call with USU = 60 B, granted units = 494 B.
  4. The current value of bucket = 4940 B (5000 – 60); the delta is 4940 B
  5. After applying SAF = 10%, the value becomes 494 B; therefore, the granted units are 494 B.
