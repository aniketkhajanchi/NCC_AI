METHOD: GET

Example:

GET
https://xxx.xxx.xxx.xxx/cctf/api/system/healthCheck

{
  "status": "Normal operation"

Change password of default account

PATH: api/user/changepassword/

METHOD: POST

Example:

POST
https://xxx.xxx.xxx.xxx/cctf/api/user/changepassword/
request body:
{
  "oldPassword": "cctf@CSF",
  "newPassword": "cctf@CSF",
  "confirmPassword": "cctf@CSF"
}
response body:
{
  "res": 0,
  "resMsg": "The password has been changed!",
  "obj": ""
}
