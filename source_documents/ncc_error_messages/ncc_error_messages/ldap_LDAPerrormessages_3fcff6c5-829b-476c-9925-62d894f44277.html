Ldap Error Messages

This table describes the LDAP error messages that the NCC may generate.

Table: LDAP error messages
Number Message HTTP Error Code Description
LDAP_00003 No mandatory properties are set. 500 Verify that the properties are set for all mandatory fields.
LDAP_00004 At least one returnable LDAP attribute must be specified. 500 At least one returnable LDAP attribute must be specified.
LDAP_00005 The value \"{0}\" entered for \"{1}\" is less than the lower limit of \"{2}\". 500 The lower limit validation of the LDAP external system configuration failed for an integer property.
LDAP_00006 The value \"{0}\" entered for \"{1}\" is higher than the upper limit of \"{3}\". 500 The upper limit validation of the LDAP external system configuration failed for an integer property.
LDAP_00007 \"{0}\" FQDN or IP address: \"{1}\" is not valid. 500 LDAP external system configuration FQDN/IP address validation failed. A valid FQDN should be resolvable by DNS. A valid IP address consists of four integers separated by a dot; for example, 138.61.45.5.
LDAP_00008 \"{0}\" with value \"{1}\" could not be converted to an integer. 500 LDAP external system configuration validation failed: cannot convert to an integer.
LDAP_00009 Name is already in use. 500 The LDAP external system Name provided is already in use for another external system.
LDAP_00010 Administrative State needs to be disabled during the configuration changes. 500 Changes can only be accepted when the Administrative State is Disabled.
LDAP_00011 Unable to establish the connection. 500 Unable to establish the LDAP connection with the given configuration data. See the logs for more information.
LDAP_00012 No external system type defined. 500 The external system type must be defined.
LDAP_00013 Invalid credentials. 500 Unable to establish the LDAP connection: Invalid credentials.
LDAP_00014 Failed to get property \"{0}\". 500 Unable to create external LDAP: Property missing.
