Tools

This section describes the tools that are available outside of the Service Manager GUI. Although most provisioning and configuration is done by using the Service Manager GUI, NCC also contains tools for other specific tasks.

The NCC database is managed by using a series of command line tools. The tools may be run by the customer or by Nokia support personnel. The tools are provided on the admincli pod, a singleton pod specifically for this purpose. A kubectl command is run by users to gain access to the admincli pod for running these tools and scripts. For example, this command is used to run the db-stats tool on a pod with name npcsps-me-admincli-68d947dc65-d5bcx on namespace sps-me:
kubectl exec -it -n sps-me npcsps-me-admincli-68d947dc65-d5bcx – /opt/tpa/bin/db-stats -s all
