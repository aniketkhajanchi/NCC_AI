Provisioning

This module is responsible for taking requests from the NCC Service Manager subsystem and performing all services, subscriber, and system-related provisioning inside the ME subsystem. It also generates EDR for various provisioning requests and, wherever applicable, generates notification events towards the NCC Notification component.
