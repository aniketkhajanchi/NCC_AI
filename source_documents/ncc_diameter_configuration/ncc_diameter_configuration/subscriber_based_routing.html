Subscriber based routing

For MVNO subscribers it is possible to redirect Diameter traffic to an external OCS. NCC identifies the MVNO based Service Provider ID and routes accordingly. That is, for those subscribers to be redirected to external OCSs, NCC keeps a record of subscribers with assigned Service Provider IDs.

To add a subscriber based routing, in the SM GUI click Connectivity → ☰ → DIAMETER ROUTES → Subscriber based Routing. Click + and configure the parameters in the following table:
Table: Subscriber based routing parameters

Parameter

Description

Optional or Mandatory
Identifier Specify an identifier for the service provider in NCC. Mandatory
Service provider ID Specify the actual service provider ID. Mandatory
