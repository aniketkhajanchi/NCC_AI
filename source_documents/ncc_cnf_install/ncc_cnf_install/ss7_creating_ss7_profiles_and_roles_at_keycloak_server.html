Creating SS7 profiles and roles at User and Role Management server

You can create profiles and roles from the profile object of the pfmaccess service in the OSP-GUI.

After creating profiles, you can create roles from the Create Role option in the Session menu.

When you click on Create Role, a pop-up window appears where you must provide inputs like Admin login, password, and a new Role name same as profile. This login and password would be of User and Role Management.

After you click on the OK button, a role creation request is triggered. On successful role creation a message is displayed on OSP-GUI as Role successfully created. For verification, you can check role from the User and Role Management server.
