Backup and Recovery for CGF (DR) MariaDB

The data in the MariaDB for CGF (DR) does not require backup and restore. If the MariaDB node fails, MariaDB data can be completely recovered from the Overrides file by reinstalling the CGF (DR) Helm chart.

Although a complete backup of MariaDB is not performed, the CDR streams backup a few configuration tables from MariaDB.
