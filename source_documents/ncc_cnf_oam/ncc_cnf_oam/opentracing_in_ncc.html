OpenTracing in NCC

These modules can be traced with OpenTracing in NCC for the N40 interface including DB client and LTM client calls.

This table describes the charging-helm chart properties that are used to enable or disable OpenTracing in NCC.
Property Description Default value
global.opentracing.enabled This property enables OpenTracing. The valid values are true or false. true
global.opentracing.span This property enables span creation. The valid values are off or on. off
global.opentracing.dbspan This property enables the Aerospike span creation. The valid values are on, off, or debug. This property is only valid if global.opentracing.span is set to on. on
global.opentracing.ltmspan This property enables LTM span creation. The valid values are off or on. off
global.opentracing.chargingspanlevel This property controls the level of N40 span creation. The valid values are in the range 1 to 5. This property is only valid if global.opentracing.span is set to on. 1
