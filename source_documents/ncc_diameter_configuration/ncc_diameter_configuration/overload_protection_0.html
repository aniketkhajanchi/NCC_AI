Overload protection

To configure overload protection, in SM GUI navigate to Connectivity → ☰ → OVERLOAD → Overload Protection. Select OVERLOAD tab. Click + and configure the overload protection parameters as described in the following table:
Table: Overload protection profile parameters
Parameter Description Mandatory or Optional
Name Specify the name of the overload protection configuration. Mandatory
Poll interval (seconds) Specify the polling time-interval in seconds. Mandatory
Escalation interval (seconds) Specify the escalation time-interval in seconds. Mandatory
