Create Helm overrides file for the Container Package Management operator

The CRDs are installed using a Helm chart. Create a crd-install-ccpm-values.yaml file with these values:

namespaced:
  enabled: true
operator:
  deploy: false
crd:
  create: true
  keepAfterUninstall: false
Note:
If the crd.keepAfterUninstall value is set to true (default value), the CRDs are preserved even if the Container Package Management operator is uninstalled.
