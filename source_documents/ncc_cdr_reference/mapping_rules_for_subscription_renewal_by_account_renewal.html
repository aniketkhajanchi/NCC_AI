maxRenewals. maxRenewalsUnlimited

 <parent>.maxRenewals

The input is is a string.

If the value can be parsed as an integer, it should be mapped into maxRenewalsValue.

Otherwise, if the value is “Unlimited”, use maxRenewalsUnlimited. The value can be anything as it will be encoded to null.

Otherwise, raise a warning and do not include in the output.
