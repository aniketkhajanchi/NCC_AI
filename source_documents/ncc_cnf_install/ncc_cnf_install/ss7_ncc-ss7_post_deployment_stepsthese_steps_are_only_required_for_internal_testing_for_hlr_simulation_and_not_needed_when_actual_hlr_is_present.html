NCC SS7 Post deployment steps

These steps are only required for internal testing for HLR simulation and not needed when Actual HLR is present.

  1. Connect to OSP PODs by using this command. 
    # kubectl exec -it -n <NAMESPACE> <POD-NAME> -- ksh -l
  2. Create SEPGROUP and SEP for HLR calls for RTCC and USSD service.
  3. Create OMNI SEPGROUP by using this command. 
    toc> pfmconfig.sepgroup.create,ACTIVE_HOST="",DOM_RF="",IDS="",NAME="omnitcap-ssn12-P1",REMARK="Service sepgroup",SWOV_FLG="4";

  4. Create dummy SEP for RTCC and USSD service with the TOCs below.

    1. -----USSD-------

      Replace the hostname (HOST_REF) as per the current host.

      toc>pfmconfig.sep.create,ARGS="",MAX_CPU="0",CPU="0",DEFCMD="",DEFMODE="",EXENAME="",GNAME="in",GRPREF00="omnitcap-ssn12-P1",HOST_REF="ss7ussd",ID="0",NAME="loadsimnumberportability_notif",N_DLGS="65000",PRIORITY="1",RATE="100",SERV_REF="ussd_20_60",SMF_REF="",STATE="0",STOPCMD="",STOPMODE="y",STRTCMD="",STRTMODE="y",UNAME="linus";

      Replace the highlighted <SEP RI> with the RI of the SEP created above.

      toc>pfmconfig.sep.modify,ARGS="",MAX_CPU="",CPU="",DEFCMD="",DEFMODE="",EXENAME="/osp/slee/service/ussd_20_60/bin/exec/ussdgw23_sep <SEP RI>",GNAME="",GRPREF00="",HOST_REF="",NAME="loadsimnumberportability_notif",N_DLGS="",PRIORITY="",RATE="",SERV_REF="",SMF_REF="",STATE="2",STOPCMD="",STOPMODE="",STRTCMD="",STRTMODE="",UNAME="";
toc>pfmomniss7.ss7_sep.create,PROTOC="1",SEP_REF="loadsimnumberportability_notif",SSN_REF="SSN12"&""&""&""&""&""&""&"",SSN="12";

    2. ------RTCC-------

      Replace the hostname (HOST_REF) as per the current host.

      toc>pfmconfig.sep.create,ARGS="",MAX_CPU="0",CPU="0",DEFCMD="",DEFMODE="",EXENAME="",GNAME="in",GRPREF00="omnitcap-ssn12-P1",HOST_REF="ss7rtcc",ID="0",NAME="loadsimnumberportability_dummy",N_DLGS="65000",PRIORITY="1",RATE="100",SERV_REF="rtcc_20_6",SMF_REF="",STATE="0",STOPCMD="",STOPMODE="y",STRTCMD="",STRTMODE="y",UNAME="linus";

      Replace the highlighted <SEP RI> with the RI of the SEP created above.

      toc>pfmconfig.sep.modify,ARGS="",MAX_CPU="",CPU="",DEFCMD="",DEFMODE="",EXENAME="/osp/slee/service/rtcc_20_6/bin/exec/RTCC19_sep <SEP RI>",GNAME="",GRPREF00="",HOST_REF="",NAME="loadsimnumberportability_dummy",N_DLGS="",PRIORITY="",RATE="",SERV_REF="",SMF_REF="",STATE="2",STOPCMD="",STOPMODE="",STRTCMD="",STRTMODE="",UNAME="";
 toc>pfmconfig.sep.modify,ARGS="",MAX_CPU="",CPU="",DEFCMD="",DEFMODE="",EXENAME="",GNAME="",GRPREF00="",HOST_REF="",NAME="loadsimnumberportability_dummy",N_DLGS="",PRIORITY="",RATE="",SERV_REF="",SMF_REF="",STATE="0",STOPCMD="",STOPMODE="",STRTCMD="",STRTMODE="",UNAME="";
 toc>pfmomniss7.ss7_sep.create,PROTOC="1",SEP_REF="loadsimnumberportability_dummy",SSN_REF="SSN12"&""&""&""&""&""&""&"",SSN="12";
  5. Update /osp/local/conf/sepgrp/sepgroup_routing.cnf, and add newly sepgroup(s) & sep RI mapping. 
    Add/update – [SEPGROUP-RI]=[SEP-RI]
