Appendix: Failure reasons

If the result tag in an EDR is Fail, then the reason tag provides a number which indicates the reason for failure as described in the following table.

Table: Possible values and description of reason tag
Value Description
0 Not Enough Funds in Account
1 Account is Barred
2 Device is Barred
3 Device is in Final State
4 Group is Barred
5 Group is in Final State
6 All Device or Group Barred or Final
10 Account overage limit is greater than the reset value
11 Invalid Discount Percentage
12 Max Renewal Reached
13 No subscription exists with align bundle reference
14 Subscription(s) with align bundle reference is (are) barred
15 Activation time is greater than or equal to aligned expiry time
16 Reference bundle subscription is not active/attached to any Device and Group
20 Max Bundle instances exceeded for Device and Group.
22 Account loan state is already OPT_IN
23 Account loan state is OPT_OUT with remaining debt
24 Account loan state is already OPT_OUT
25 Multiple device found
26 No devices match given identities
27 Account cannot be found as device has no subscription
28 Account doesn't exist
29 Loan is rejected as per configuration by customer
30 Usage limit exceeded
31 Mandatory Subscription is Suspended
32 Renewal End date reached
