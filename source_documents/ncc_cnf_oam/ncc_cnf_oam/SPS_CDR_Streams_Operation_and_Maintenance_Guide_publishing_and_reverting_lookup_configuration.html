Publishing and reverting lookup configuration

Lookup tables are used for managing the configuration of distribution rules through Charging Gateway Function UI. After modifications, lookup tables must be published to the database for the configuration changes to take effect.

The lookup configuration currently in use (published) can be seen as read-only lookup tables with suffix P_ in the table names. The corresponding editable draft lookup tables have the same name but without the prefix. If necessary, it is also possible to revert a draft lookup table back to its previous published state.

For more information, see the Overview Guide.
