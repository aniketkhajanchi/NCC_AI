Configurations

Log Export supports the following environment variables.

ENV Default Description
LOG_INDICES '["log-*", journal]' An OpenSearch based simple query to control the index-pattern to list.
DEFAULT_FIELDS "log,message" Comma separated values. Default search fields to search upon when field key is not provided in search query.
TIMESTAMP_FIELD '@timestamp' Default time field. Applicable to all indices.

Use these variables for tuning. Change these variables only to address download speed issues.

ENV Default Description
EXPORT_CHUNK_SIZE "500" Tune stream chunk size for exporting (number of records in one stream). Higher number can also clog slow down.
SCROLL_TIME "10m"

OpenSearch API scroll value.

This query is used for creating stream chunk, refer diagram.
EXPORT_TIMEOUT "40s" Timeout for above scroll based query SCROLL_TIME.
HIGH_DOWNLOAD_LIMIT "200000" App pops a confirmation to user when size of record to download reaches this value to discourage download of huge records.
HIGH_WATER_MARK_BROWSER "2" highWaterMark for browser
HIGH_WATER_MARK_SERVER "5" highWaterMark for server

The highWaterMark option is a threshold, dictating the amount of data that a stream buffers before it stops asking for more data. The option does not enforce a strict memory limitation in general. Specific stream implementations may choose to enforce stricter limits, but doing so is optional.
