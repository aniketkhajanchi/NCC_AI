Get-Match-String ALL

RESOURCE_PRE_SORTING

 This attribute checks whether a string matches with another string. If the string matches, it returns true. Else, it returns false. It takes the following input parameters:
  • Input-String1: The first string (string1;string2;string3).
  • Input-String2: The string to be matched with the first string (string1).
  • Separator (;): It separates the strings in the Input-String1.
For example: CALL_COMMON.Get-Match-String("string1;string2;string3", "string1", ";")
