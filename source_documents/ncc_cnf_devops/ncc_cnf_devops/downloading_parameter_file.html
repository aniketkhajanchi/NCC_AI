Downloading parameter file

It is possible to download the parameter file of an existing test package in CCTF.

  1. Click the three-dot button besides the name of the test package to be downloaded, for example:
  2. Select Download Parameters File from the list.
    Note:
    When downloading the parameters of a test package through the CCTF UI, the value for each item in the sensitive_data section will be masked in the UI leaving the key of these items unmasked.
