Value mapping (case-insensitive):

  • Volume -> 0 (volume)
  • Money -> 1 (money)
  • Time -> 2 (time)
  • Unit -> 3 (unit)

Validate that the input is of type Real. Raise a warning and do not map if it is not.

Value mapping (case-insensitive):

  • False -> 0 (false)
  • True -> 1 (true)

Do the conversion as described in Mapping of Real values.
