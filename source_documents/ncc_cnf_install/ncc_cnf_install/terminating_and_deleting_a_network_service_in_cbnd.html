Terminating and deleting a network service in NCOM

This section provides information on how to terminate and delete a network service in NCOM. To delete a network service, you first need to terminate this network service successfully.

Terminating a network service

When you terminate a network service, NCOM deletes a network service and removes all the resources associated with the Helm charts deployed in the NCS cluster. This means that Helm charts are uninstalled. For information on how to terminate an network service, see section Managing Network Services/Terminating an NS in <NCOM Version x> Operating Guide.

Deleting a network service

If the termination of a network service is successful, you can delete this Network Service. In this event, all information about this NS on NCOM is also deleted. The resources on NCS cluster were removed during Terminate Lifecycle event. For information on how to delete an network service, see Managing Network Services/Deleting an NS section in <NCOM Version x> Operating Guide.
