if

    SPR_ACCOUNT.Overage-Limit = 0

Secondary-Balance

SUBSCRIBER_POST_PROVISIONING

SUBSCRIBER_PRE_PROVISIONING

SUBSCRIBER_POST_TRANSFER

SUBSCRIBER_PRE_TRANSFER

THRESHOLD

CALL_RULE_VARIABLES

SUBSCRIBER_FAILED_PROVISIONING

 Secondary Balance is used as part of the rules and available in the notifications.

The following values are available:

  • Bucket-Name: The name of the bucket used while rating a call.
  • Value: This attribute provides the value.
  • Initial-Value: This attribute provides the bucket initial value.
  • End-Date: This attribute provides the end date of a periodic lifecycle attached with an account.
An index shall enable to identify the specific Secondary-Balance value.

Example: Secondary-Balance.Bucket-Name ( x), Secondary-Balance.Value ( x). If Secondary Balance is not present for an index x (where x value can be 0, 1, 2...), the return value is NULL.
