SFTP server and connection parameters

The address of the SFTP server is configured with node parameters SFTPHost and SFTPPort. The target directory on the SFTP server is defined with node parameter SFTPRemoteDirectory.

Authentication on the remote server is configured using node parameters SFTPUsername, SFTPPassword and SFTPPrivateKey, which specify the local path to the private key file. The encryption and compression algorithms are specified with node parameters SFTPCipher, SFTPKex, SFTPMac and SFTPCompression.
