Configurations in SM GUI

The URL of the NCC SS7 Web UI Application must be configured in the NCC SM GUI in the field SS7 URL of Application Preferences under Global Configuration.

This URL is in this format:

https://$FQDN/ss7manager

$FQDN is the namespace TLS FQDN.
