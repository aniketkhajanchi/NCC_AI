Modifying existing parameter file

The following procedure describes how to modify the data items in the parameter file through the CCTF UI.

  1. Click the three-dot button besides the name of the test package to be updated, for example:
  2. Select Edit Parameters from the list.

    A Parameters Editor view is displayed with the original values of all parameters, for example:

  3. Edit or update any parameters as needed.
    Note:
    When editing the parameters of a test package through the CCTF UI, the value for each item in the sensitive_data section will be masked in the UI leaving the key of these items unmasked. String "**" is not allowed as the value of any data in the sensitive_data section.
  4. Click SAVE to save the changes.
