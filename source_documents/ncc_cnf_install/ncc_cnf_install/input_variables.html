Input variables

Input values can be supplied to terraform by using the tfvars syntax. For more information on the syntax, see https://www.terraform.io/language/values/variables.

Table: Input variables
Attribute name Default value Mandatory Type Description
aws-profile null yes string Name of the aws profile you have setup to connect to your aws account.
region “us-east-1” no string AWS region to be used.
permissions-boundary “arn:aws:iam::966096131882:policy/NokiaBoundary” no string Permission boundary if any for your account.
