policy-charging-plugin-sequence

Description

This command provides a list of the active plugin sequence on current charging node. The tool is available at this path:

/opt/tpa/bin/policy-charging-plugin-sequence.

Syntax

  • -l (syntax: policy-charging-plugin-sequence -l): Run on current charging node, returns current active ddm plugin sequence.
  • -h (syntax: policy-charging-plugin-sequence -h): Help option
