Configuring distribution rules

Distribution rules define the destination of different output records. The configuration is maintained through lookup tables in the GUI. For File output stream, distribution rules define which output records are stored in which files and which directories. For Kafka output stream, distribution rules define which topic to write data to.

For more information on distribution rules, see the Overview Guide in the NCC documentation set.
