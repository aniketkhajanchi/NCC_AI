Tag Description Example
taxationID Indicates the ID of the Taxation Profile associated with the entity. tax1
taxRatePercentage Specifies the percentage rate of tax [15, 10.25, 5]

taxationLastUpdateTime Time at which the Taxation Profile was last modified. 06/07/2021 12:00:33
applicationType Specifies the application type. CHARGING
