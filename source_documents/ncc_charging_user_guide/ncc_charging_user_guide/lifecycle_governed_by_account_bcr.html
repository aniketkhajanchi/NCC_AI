Subscriptions governed by account PLC

The following lifecycle configurations are applicable to subscriptions governed by account PLC (that is, when the application preference Controlled Renewal Sequence is set to VIA_ACCOUNT).

If Controlled Renewal Sequence is set to ALL_SUBSCRIPTIONS, then the following lifecycle configurations are applicable only to subscriptions governed by account PLC.
