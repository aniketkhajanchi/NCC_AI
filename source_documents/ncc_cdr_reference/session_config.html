Session management configuration

EDR tags

Tag Description Example
sessionManagementConfigId Indicates the ID of the Session Management Configuration associated with the entity. SessMgmt1
creationTime The timestamp at which the entity is created. 06/07/2021 12:00:33
lastUpdateTime Time at which the Session Management Configuration was last modified. 06/07/2021 12:00:33
SySessionTermination This is record property for Sy Session Termination information. SySessionTermination
N28SessionTermination This is record property for N28 Session Termination information. N28SessionTermination
enableSessionTermination+ To enable initiation of Sy session termination true
maxSession+ Specifies the maximum number of Sy sessions allowed per device 3
pcrfNames Specifies the list of PCRF nodes configured. Present only for SySessionTermination [PCRF1, PCRF2, PCRF3]
N40SessionAttribute This is record property for N40 Session information. N40SessionAttribute
RoSessionAttribute This is record property for N40 Session information. RoSessionAttribute
GySessionAttribute This is record property for N40 Session information. GySessionAttribute
maxActiveSessionLimit++ Specifies the maximum number of N40 sessions allowed per device 5
auditSessionLimit++ Specifies the audit limit for N40 sessions allowed per device 3
validityTime++ Specifies the VT defined for N40 session when audit limit is reached 300
