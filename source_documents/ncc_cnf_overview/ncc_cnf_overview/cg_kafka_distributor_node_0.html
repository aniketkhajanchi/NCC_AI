CG Kafka Distributor Node

This node is the same node that is used in the Kafka output stream. For details on the node, see CG Kafka Distributor Node.

Redelivery stream record format

The redelivery stream aggregates records that stored in the SFTP directory into batches and stores each batch in a database. The number of records in a batch is defined with the CG Kafka Failure Collector Node parameter RecordsPerBatch. Inside the batch, records are stored using the format in which they were received by the Business logic stream. The files can be stored either as they are or in a compressed format.
