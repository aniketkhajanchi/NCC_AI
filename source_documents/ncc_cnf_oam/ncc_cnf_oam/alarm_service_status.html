Alarm service status

The alarm service is a container which resides on the Alarm Manager pod. These are some common ways to check the alarms service status.
  • The Alarm Manager pod logs may be checked using this command.
    kubectl logs -f -n <namespace> <pod name> <container name> --tail=10
  • The existence of the Alarm Manager pod may be verified with this command.
    kubectl get svc -n <namespace> | grep calm
