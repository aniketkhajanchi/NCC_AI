Downstream overload profile

Downstream overload protection profiles capture the overload parameters for egress peers and end destinations. You can add and customize as many message filters as necessary for specific peers and end destinations depending on the desired response.

To configure a downstream overload profile, select PROFILES tab in the Downstream Overload window. Click + or CREATE NEW DOWNSTREAM OVERLOAD PROFILE and configure the parameters as described in the following table:
Table: Downstream overload profile parameters

Parameter

Description

Mandatory or Optional
Name Specify the profile name. Maximum characters 50 only. Mandatory
Description Specify a description of the profile. Optional
