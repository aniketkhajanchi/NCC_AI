Variables for resource-based counter

The following context variables can be used for resource-based counter:
  • Reserve-Rate: This variable indicates the Rate ID with the entity for which the maximum reservation has taken place.
  • First-Reserve-Rate: This variable indicates the Rate ID with the entity that is first used in reservation.

See Determining rate based on counter threshold to get the Rate ID.
