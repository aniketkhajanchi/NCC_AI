Mandatory parameters

The parameters mandatory for each test package are located in the test_package_info section in the parameters.yaml file. They provide the basic information on the test package. The following table presents these mandatory data items, and whether the data item must be defined, that is, Mandatory (M) or Optional (O).

Note:
All data items in the test_package_info section must remain even if having no value.
Table: Mandatory CCTF parameters
Data item Description Example value M/O
platform_name The platform name of the product. NCC O
platform_ver The version number of the platform. 1.0 O
product_name The name of the product to be verified by this test package. NCMS O
product_ver The version number of the product. 1.0 O
runTime_OS_name The operating system of the run time environment. CentOS O
runTime_OS_ver

The operating system version number.

Default value: 7.8

 7.8 O
testPackage_name

The name of this test package.

Default value: spstests

 spstests O
testPackage_ver The version of this test package. 1.0 O
