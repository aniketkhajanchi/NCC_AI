Auditing

All the streams use the standard Charging Gateway Function auditing interface, meaning that counters and throughput can be monitored at node level. The audit information captured for each specific node is detailed in the following sections.

In the following tables, the Counter type column indicates the types of counters that are incremented by records coming into the node (left of the = sign) and by outgoing records (right of the = sign). Counter types available are:
  • RecordsIn
  • RecordsOut
  • Filtered
  • Rejected
  • ReprocessedIn
  • Created
  • Duplicated
  • Stored
  • Retrieved
  • ReducedInMerging
