Enabling mapping of CHF records

To enable mapping of CHF V15 and V16 records, the following configuration changes are required in CGF.

  1. In the Business logic stream, set CG Mapper Node node parameter CHFMappingEnabled to yes.
  2. If release 15 mapping is required, modify the node configurations as follows:
    • In Business logic stream, set CG Mapper Node node parameter CHFMappingVersion to 15.
    • In File output stream and Kafka output stream, set ASN.1 Encoder Node node parameter DescriptionFilename to description.asn.
  3. If release 16 mapping is required, modify the node configurations as follows:
    • In Business logic stream, set CG Mapper Node node parameter CHFMappingVersion to 16.
    • In File output stream and Kafka output stream, set ASN.1 Encoder Node node parameter DescriptionFilename to description_v16.asn.
