About this document

The Diameter/HTTP Configuration User Guide provides information about Diameter and 5G HTTP configuration and usage in NCC. The information available in this guide often refers to the use of rules, rule system versions (RSVs), and message filters. See the Charging User Guide for generic information about working with rules and RSVs.

The Diameter HTTP Configuration application is now available as Connectivity in the new GUI. This document has been update in line with the changes for the new GUI. However, there may be some references to old navigation, which will be updated in the next release.
