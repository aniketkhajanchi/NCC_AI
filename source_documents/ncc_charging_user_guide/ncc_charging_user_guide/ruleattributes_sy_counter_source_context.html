SY_COUNTER source context

In SNA_MANAGEMENT trigger, the SY_COUNTER source context is reserved for future use.

Attribute name Trigger Description
Is-First-Use-Counter SNR_RESULT This attribute checks whether counter is used for the first time.
SNA_MANAGEMENT

SNA_MANAGEMENT
Policy-Counter-Id-Old SNR_RESULT This attribute provides the old policy counter id.
SNA_MANAGEMENT
Policy-Counter-Status-New SNR_RESULT This attribute provides the signaling state of counter from Sy result answers.
SNA_MANAGEMENT
