Uninstalling Helm charts

This section provides information on how to uninstall the Helm charts.

Each site's examples directory contains an example uninstall.sh. This script attempts to remove all charts and ignores errors for a quick uninstall. It can also remove the RBAC resources, sidecar, PVCs, and namespace.

To execute uninstall script, enter:
cd output/SITENAME
  
# Default execution with no arguments only removes charts:
./examples/uninstall.sh
 
# Execute with the following argument to remove everything including PVC and namespace:
./examples/uninstall.sh destroyEverything

If you created additional or optional resources, to delete them as well, follow the same syntax as shown in the example script.
