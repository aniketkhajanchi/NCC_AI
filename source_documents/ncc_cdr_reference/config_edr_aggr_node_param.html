Configuring EDR aggregation node parameters in CDR streams

For EDR aggregation, set the following values for node parameters:

  • For CG Kafka Collector Node (Kafka input stream):
    • PassThrough: no (hidden)
    • SessionIdFields: sessionId,recordSequenceNumber,correlationID
    • SequenceNumberFields: recordSequenceNumber,aggSequenceNumber
    • EventRecordTypes: PROVISIONING_RECORD,RechargeRecord
    • ChargingRecordTypes: make it empty
    • SessionTimeout: To keep a session ongoing in Redis, set this node parameter to a value greater than the validity time on the NCC side.
  • For CG Mapper Node (Business logic stream):
    • RecordSequenceNumberValidation: yes (hidden)
