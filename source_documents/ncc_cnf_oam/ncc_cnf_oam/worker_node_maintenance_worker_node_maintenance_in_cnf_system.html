Worker node maintenance

This section describes the procedure to gracefully remove a worker node for maintenance operations.

Note:
This section assumes the NCC system is fully functional, and that the backup-restore tool is not currently running. Also note that the procedure may not be applicable to edge nodes, and any specific values that are given are only examples and should not be taken literally. The system must have enough space to allow the pod movements from one node to another.
