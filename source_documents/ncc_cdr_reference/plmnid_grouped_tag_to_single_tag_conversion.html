plmnId grouped to single conversion

In 5G, PLMN ID is an object and NCC converts it into an octet string. Bits in octet string depend on MCC and MNC values as defined in the following tables. The type is defined as per CDR description in 3GPP TS 32.298 V16.2.0 (2019-09).

Table: Octet/bit position
Octets/Bit Position 7 6 5 4 3 2 1 0
1 MCC digit 2 MCC digit 1
2 MNC digit 3 MCC digit 3
3 MNC digit 2 MNC digit 1
Table: Octet/bit position if mnc does not have a third digit
Octets/Bit Position 7 6 5 4 3 2 1 0
1 MCC digit 2 MCC digit 1
2 1111 MCC digit 3
3 MNC digit 2 MNC digit 1

5G representation: PLMNId object:

nFPLMNID = { mcc:"123", mnc:"456" }

nFPLMNID = { mcc:"123", mnc:"56" }

CDR Representation:

networkFunctionPLMNIdentifier :: 0x216354

networkFunctionPLMNIdentifier :: 0x21F365
