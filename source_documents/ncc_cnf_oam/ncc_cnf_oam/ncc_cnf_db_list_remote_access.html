Access db-list output files remotely

The output of db-list commands are stored in /opt/tpa/dump in the admincli pod. It is possible to access these files remotely through an SFTP server which is enabled in the admincli pod.

Configure the admincli SFTP server

The admincli SFTP server can be configured by editing the values for the inputs Admincli SFTP xxx in the SPS section of the Common worksheet of the Inputs.xslx file that is used to generate the overrides for your deployment.

If both Admincli SFTP Password Enabled and Admincli SFTP SSH Key Enabled are set to false, the SFTP server is not accessible. Enable both of these values to use just the password to login.

The Inputs.xlsx spreadsheet shows the default values assigned to the private/public keys for the admincli SFTP server and the password that must be used to login.

Login to the SFTP server

The admincli SFTP server is listening on port 2310. To connect to the SFTP server, use these commands and provide the password when prompted. 
sftp -P 2310 sftp@site1-bcmt2-dk.dyn.nesc.nokia.net
                    sftp@site1-bcmt2-dk.dyn.nesc.nokia.net's password:      // enter your password if you changed it
                    Connected to site1-bcmt2-dk.dyn.nesc.nokia.net.
                    sftp> ls      
                    sftp> exit

Generate new keys and password

To generate new keys to override the default ones and encrypt a new password, run these commands: 
# Generate the keys using openssl
openssl genpkey -algorithm RSA -out private_key.pem -pkeyopt rsa_keygen_bits:2048
openssl rsa -pubout -in private_key.pem -out public_key.pem
                    
# Encrypt the password by RSA (the password in this example is "foo@123"). The password has to be at least 8 characters and has to include a special character
echo -n "foo@123"|openssl rsautl -encrypt -pubin -inkey public_key.pem -out >(base64 -w 0)
                    
# The output of the echo command is the encrypted and encoded password, paste this text to the spreadsheet as the value for “Admincli SFTP Password”
                    
# Also, copy the content in private_key.pem to “Admincli SFTP Password Private Key”
# and the content of public_key.pem to “Admincli SFTP SSH Public Key”.
