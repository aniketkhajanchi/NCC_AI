Kafka output stream

Kafka output stream collects batches of output records from the database. It encodes the data in ASN.1 format and distributes the encoded records to Apache Kafka topics, according to pre-configured distribution rules (see Distribution Rules). If the stream fails to store some records (for example, due to unavailable Kafka brokers), it forwards them to the Log distributor stream, which writes the records to the error log.

Kafka output stream contains the following three nodes:

  • CG Collector Node – a node that reads a batch of output records from the database and sends them forward in CGF (DR) internal format.
  • ASN.1 Encoder Node – a node that converts the batch of output records into a file in ASN.1 format.
  • CG Kafka Distributor Node – a node that distributes the encoded records to Apache Kafka topics.
