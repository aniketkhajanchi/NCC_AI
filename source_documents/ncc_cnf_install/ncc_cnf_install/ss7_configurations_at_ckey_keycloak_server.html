Configurations at User and Role Management Server

Note:
It is assumed that the User and Role Management server is already configured for use with NCC SM GUI.
Note:
All configurations need to be done in the realm that is already configured on the User and Role Management server for NCC.

These one-time configurations need to be done at the User and Role Management server to enable NCC SS7 to be authenticated through User and Role Management, and for the NCC SS7 Web Application to be integrated with NCC SM GUI through SSO.
