LOAN source context

Table: LOAN source context
Attribute   Description
Loan-Id LOAN_POST_PROCESSING This attribute defines ID of loan.
Loan-Amount LOAN_POST_PROCESSING This attribute defines loan amount.
Loan-Bundle-Name LOAN_POST_PROCESSING Reference to bundle definition.
Loan-Activation-Fee LOAN_POST_PROCESSING This attribute defines loan activation fee, that is, fee to be deducted during opt-in.
Loan-Service-Fee LOAN_POST_PROCESSING This attribute defines Service fee of loan, that is, fee to be deducted during payback.
Type-Of-Loan LOAN_POST_PROCESSING This attribute defines loan type.
Loan-Pay-Service-Fee-If-Loan-Used LOAN_POST_PROCESSING Fee is paid only if loan used. Service fee is deducted on the basis this parameter. Its default value is true.
Transaction-Id LOAN_POST_PROCESSING It indicates the transaction Id.

It is taken from the loan request. If it is not present in the loan request, then it is generated internally in UUID format.
