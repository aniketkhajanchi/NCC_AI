Charging Step

EDR tags

Tag Description Example
chargingStepName Name of charging step. step94203
↘stepSize* Size of a step. 5.00
↘stepFee* The fee to be deducted on step up. 2
chargingStepCreationTime Specifies the time at which the entity was created. 13/09/2019 09:37:07
chargingStepLastUpdateTime Specifies the time at which entity was last modified. 13/09/2019 09:37:20
chargingStepUnitType Indicates the unit type for the entity. MByte
chargingStepKindofUnit Indicates the kind of unit for the entity. VOLUME
taxationID Specifies the identifier for the taxation profile. tx03
taxIncluded This parameter indicates whether applied tax is included to the fee/cost or not.

If true, tax is included to the cost/fee.

If false, tax is applied over and above cost/fee.

Default Value: false

 true
lastStepUnlimited Indicates if last step is unlimited or not. Possible values: true, false. false
