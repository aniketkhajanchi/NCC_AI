Table: GY_MESSAGE result context
Attribute name Trigger Description
Add-MSCC GY_SESSION_ESTABLISHMENT This action is required to add the preemptive categories at Call Establishment, so that they can be used at charging Triggers.

GY_MESSAGE.Add-MSCC (<MSCC-Type>, <List of strings>)

MSCC-Type: is an ENUM with the following values: Rating-Group_ENUM or Service-Identifier_ENUM (currently only Rating-Group is supported)

List of strings: is a list of strings, which will be used to add a new MSCC, with the Rating-Group information AVP filled with the given value.
