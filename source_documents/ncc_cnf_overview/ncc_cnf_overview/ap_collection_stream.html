Assume Positive collector stream

The Assume Positive collector stream consists two nodes, Assume Positive Collector Node and Assume Positive Filtering and Distributor Node.

Assume Positive Collector Node collects compressed or uncompressed raw data files from SMF hosts, parses the files into a series of records after successful collection and sends the records into an Apache Kafka topic, to make them available for further processing by Assume Positive handler stream. The SMF hosts from which data files are collected are defined using connection points. The Apache Kafka topic is defined with node parameter KafkaTopicName.

In cases where a compressed file is corrupted during the transfer and cannot be extracted, or the input files transfer fails, Assume Positive Collector Node discards the files and places them into the directory for discarded files, to make it possible to try manual reprocessing of the data if necessary.
