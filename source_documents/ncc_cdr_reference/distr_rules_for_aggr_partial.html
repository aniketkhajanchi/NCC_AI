Distribution rules for aggregated and partial outputs

EDR aggregation will result in getting separated files of partial and aggregated outputs. Partial outputs are NCC requests populated in a generic format without any CGF (DR) processing.

Aggregated and partial files are distributed to different destinations (directory/Kafka topic).

Table: Distribution rules for aggregation output files
Description Criteria Priority Rule Destination: File Destination: Kafka
Directory File group Topic
Partial records for EDR session EL_PARTIAL_EDR = true 56 edr/partial .EDR.PARTIAL edr-partial
Final aggregated records for EDR session EL_FINAL_EDR = true 57 edr/final .EDR.FINAL edr-final
