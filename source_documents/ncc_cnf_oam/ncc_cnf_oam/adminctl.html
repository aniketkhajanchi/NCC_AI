adminctl

Description

Use this tool to run various operations (such as listCerts, updateSecret) from the admincli container.

The script performs various operations including:

  • listCerts: List all of the certificates present in secrets in the namespaces.
  • updateSecret: Update the existing secrets with new certificates.

Syntax

adminctl [-h] [-l | -o ...]
Option Description
No arguments Displays usage/help information.
-h or -? Displays usage/help information.
-l, --list List available operations.
-o ..., --operation ... Operation to perform. To list operation help : adminctl -o <operation> -h.

Examples

List all available operations.

Command: adminctl -l

Output: 

Available Operations:

listCerts : This operation provides Kubernetes users a way to easily monitor their certificates stored in secrets in different namespaces,
 ensuring that they remain up-to-date.

updateSecret : This operation updates the certificate stored in a secret that is associated with Kubernetes namespace.
 The updated certificate data is passed as a string parameter. The method first retrieves the existing secret data and updates the certificate
 field with the new data. Then it applies the updated secret to the Kubernetes cluster using the Kubernetes API implemented in Java.

To perform an operation.

Command: adminctl -o listCerts -o json

Output:
[
