Deleting test package

The following procedure describes how to delete a test package in the CCTF UI.

  1. Select and expand the target product in the product list.
  2. Click the three-dot button besides the name of the test package to be deleted, and click Delete, for example:

    A confirmation pop-up is displayed.

  3. Click DELETE to delete the test package.
    Note:
    The test package is removed permanently from the system.
