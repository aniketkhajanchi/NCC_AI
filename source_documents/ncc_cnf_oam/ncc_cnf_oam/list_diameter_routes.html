list-diameter-routes

Description

Use this tool to list the Diameter routes configurations that are applicable (that is, they are active and their referenced peer is connected). The tool is located in the /opt/tpa/bin directory and must be run by the tpaadmin user.
Note:
This tool does not work when sites are in a MOOS/OOS state or charging/IOHD pods are not READY.

Syntax

list-diameter-routes [-d] [-h]
Table: Options
Option Description
No arguments List the Diameter routes configs that are applicable.
-h or -? Displays usage/help information.
-d Include details of the routes that are not currently applicable, either because the route configuration is inactive or the peer associated with the route is not connected.
