Overload reduction metric

To configure an overload reduction metric, select Overload reduction metric tab in the Overload Protection window. Click + or CREATE NEW OVERLOAD REDUCTION METRIC and configure the parameters as described in the following table:
Table: Overload reduction metric parameters

Parameter

Description

Mandatory or Optional
Name Specifies the name of the Overload Reduction Metric. Mandatory
Minor state (%) Valid range: 1 to 100
Note:
Minor State value must be less than the Major State value.
 Mandatory
Major state (%) Valid range: 1 to 100
Note:
Major State value must be less than the Critical State value.
 Mandatory
Critical state (%) Valid range: 1 to 100
Note:
Critical State value must be less than the Resource Critical State value.
 Mandatory
Resource critical state (%) Valid range: 1 to 100 Mandatory
Validity interval (s) Specifies the validity interval. Default value: 60 seconds. Optional
