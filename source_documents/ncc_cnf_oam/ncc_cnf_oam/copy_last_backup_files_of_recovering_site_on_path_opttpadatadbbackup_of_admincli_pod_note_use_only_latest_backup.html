Copy backup files

Copy the latest backup files of the recovering site to the folder /opt/tpa/data/db-backup/ of the admincli pod.

kubectl -n <namespcae> -c <container> cp db-backup/ <admincli>:/opt/tpa/data/db-backup # we are assuming that you have copied you db-backup to <local-dir>/db-backup  .ex /root/db-backup/

# Confirm the backup is copied to the admincli pod of the recovering site.
kubectl -it exec <admicli> -c <container> -n <namespace> --   ls -lRh /opt/tpa/data/db-backup/db-backup
Where:
  • <admincli> is the name of the admincli pod.
  • <container> is the name of the container.
  • <namespace> is the name of the namespace.
