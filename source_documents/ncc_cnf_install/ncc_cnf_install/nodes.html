Nodes

Nodes is another array of objects you can pass in to create managed node groups or worker nodes for your EKS cluster.

Attribute name Default value Mandatory Type Description
name null yes string Name of your managed node group.
subnets null yes list of strings subnets ids used for the nodes(should be a subset of the cluster subnets)
ami_type AL2_x86_64 no string ami family type used for node architecture.
disk-size 40 no number default size in GB for ephemeral disk volume.
instance-types t3.2xlarge no list of string list of instance types in AWS to be used for your nodegroup.
min-size 2 no number minimum size of your node group.
max-size 5 no number max size of your node group.
capacity-type SPOT no string either ONDEMAND or SPOT as per AWS doc.
max-unavailable-percentage 25 no number How much nodes can be gracefully shutdown/recycled.
labels null no map of string labels for Kubernetes nodes.
taints null no map of object taints for nodes.
ec2-ssh-key null no string SSH key pair name on AWS.
