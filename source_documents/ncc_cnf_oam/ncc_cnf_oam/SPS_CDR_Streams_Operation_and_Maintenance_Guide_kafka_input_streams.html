Kafka input stream

Table: Kafka input stream auditing
Node Counter type Details
CG Kafka Collector Node RecordsIn = Stored + Filtered

If the input record cannot be processed for some reason (for instance decoding error), the Filtered counter is incremented. Otherwise it is marked as Stored.

The RecordsIn counter indicates the number of records coming in.
