Loan

EDR tags

Tag Description Example
loanId This parameter indicates the Id of the loan. IoanId1
typeOfLoan This parameter indicates the type of loan RECURRENT
loanAmount This parameter indicates the loan amount. 5.0
activationFee This parameter indicates the activation fee. 2.0
serviceFee This parameter indicates the service fee. 1.0
payFeeIfLoanUsed This parameter indicates if fee should be paid for loan usage. true, false
updatedServiceFeesApplicability Indicates when the service fee is applied if service fee is updated after the fee is charged from prepaid balance:

Possible values: IMMEDIATELY, NEXT_OPT_IN

 IMMEDIATELY
loanCreationTime This parameter indicates the time at which the loan was created. 08/02/2022 10:07:18
loanLastUpdateTime This parameter indicates the time at which the loan was recently updated. 08/02/2022 10:15:00
reason

If the result is fail, then this attribute provides the reason for failure.

Valid values: Refer to Appendix: Failure reasons.
