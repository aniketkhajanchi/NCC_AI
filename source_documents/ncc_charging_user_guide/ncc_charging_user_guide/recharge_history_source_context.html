RECHARGE_HISTORY source context

Table: RECHARGE_HISTORY source context
Attribute name Trigger Description
Cumulative-Recharge-Amount

LOAN_PRE_PRCESSING

LOAN_POST_PROCESSING

This attribute provides the sum of recharge amount for the recharges done in a configured period.

It takes StartDateTime, and EndDateTime as arguments.

It returns -1 if either Max Number of Recharge Histories, or Recharge History Period (days) application preference is 0.

Last-Recharge-Date-Time

LOAN_PRE_PRCESSING

LOAN_POST_PROCESSING

This attribute returns the date of the previous recharge.

It returns zero (0) if there is no occurrence of recharge.

It returns -1 if either Max Number of Recharge Histories, or Recharge History Period (days) application preference is 0.
