CG Kafka Collector Node fails to connect to Kafka broker: Timeout expired while fetching topic metadata

This error indicates that an error has occurred in the server side of Kafka broker, preventing CG Kafka Collector Node from fetching topics for validation purposes.

First, make sure that Kafka and Zookeeper services are running.

To debug this issue and find the exact failure, you need access to the server side log of the Kafka broker. Usually Kafka logs are located in /etc/kafka/logs.
