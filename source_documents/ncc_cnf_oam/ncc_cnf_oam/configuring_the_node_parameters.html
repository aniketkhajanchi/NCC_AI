Configuring the node parameters

Before using the Rest Handler node, the node parameters must configured correctly by using the CGF (DR) UI or by editing the configuration stream overrides file that is used by the input generator. This table shows the list of node parameters that must be configured.
Table: Rest Handler node parameters
Name Default Description
ConfigStreamHost   Host address of configuration stream (ME/RT FQDN)
DRDatabasePassword   Nokia Mediation database password.
DRDatabaseURI   JDBC connection string to the Nokia Mediation database.
DRDatabaseUser   Nokia Mediation database user.
HttpPath /cg/api/v1/lookup-service API endpoint HTTP path.
HttpPort 8088 Http server port.
KeycloakHost   User and Role Management host URL. ( User and Role Management FQDN )
KeycloakReam   User and Role Management ream used for this service.
SSLCertPath   HTTPS SSL certificate file path.
SSLPrivateKeyPassword   Password that protects SSL private key.
SSLPrivateKeyPath   HTTPS SSL private key path.
TrustedDrCertPath   DR certificate path
TrustedKeycloakCertPath   User and Role Management certificate path if User and Role Management uses a different certificate than used by DR
UseHttps no Using HTTPS connection.
