CGF (DR)

This module is responsible for receiving CDR/EDR records from other NCC components and performing these operations:
  • write the CDR/EDR records in files as per the generic ASN
  • manage and control the file writing based on configuration parameters
  • sends CDR/EDR to external Kafka
  • generates CDR for a specific 5G N40 interface as per ASN that is defined in 32.298
  • aggregates the CDR for data sessions

For more information about this module, see the CDR/EDR Reference and the Charging User Guide.
