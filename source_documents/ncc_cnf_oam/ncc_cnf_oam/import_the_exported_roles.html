Import the exported roles

After the migration to SSO/LDAP is completed, these steps can be performed to import the roles that you exported prior to the migration. These steps only have to be completed on one of the SM/IG systems. The User and Role Management in the other systems replicates the data from this site.
  1. Login to the User and Role Management GUI and for sps-auth realm.
  2. Click the Import option under the Manage section.
  3. Select the sps-auth-realm.json file that was exported for the Partial Import.
    1. Set Import clients to OFF.
    2. Set Import realm roles to ON.
    3. Set Import client roles to ON.
    4. Choose the appropriate option for If a resource exists. The possible values are: Skip, Overwrite, and Fail.
  4. Click Import.
  5. Repeat the same partial import for the btel-auth realm with the btel-auth-realm.json that was exported.
