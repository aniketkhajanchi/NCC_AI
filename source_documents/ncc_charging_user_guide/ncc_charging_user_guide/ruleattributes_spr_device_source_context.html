SUBSCRIBER_PRE_PROVISIONING
This attribute takes the following parameters:
  • Input-String: It consists of bundles separated by the separator.
  • Separator (;): It separates the bundles in the input string.

For example: Is-Bundle-In-ActiveSubs("bundle1;bundle2;bundle3", ";")

This attribute checks whether the bundles in Input-String are in active subscription list in the current device.

If at least one bundle from the bundle list provided in the input string is present in the active subscription list of the device, then this action returns the matched bundle name. Else, it returns null.
