list-diameter-peers

Description

Use this tool to list the Diameter peer information from the active IOHD and Charging pods. The tool is located in the /opt/tpa/bin directory and must be run by the tpaadmin user.
Note:
This tool does not work when sites are in a MOOS/OOS state or charging/IOHD pods are not READY.

Syntax

list-diameter-peers [-h] [-s] [-a]
Table: Options
Option Description
No arguments List the Diameter peer information from active IOHD node.
-h or -? Displays usage/help information.
-s Display a summary of the peers.
-a Collects information from active IOHD and Charging pods.
