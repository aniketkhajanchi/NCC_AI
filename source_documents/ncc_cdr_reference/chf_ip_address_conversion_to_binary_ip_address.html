IP address conversion to binary IP address

An IPv4 address in the input is a 32-bit number written in decimal digits and formatted as four 8-bit fields separated by periods. Each 8-bit field represents one octet (one byte) of the address. For example, the input 192.10.2.2 becomes 0xc00a0202 as octet string output.

An IPv6 address in the input is a 128-bit number written in hexadecimal digits and formatted as eight 16-bit fields separated by colons. Each 16-bit field represents two octets (two bytes) of the address. For example, the input 2001:0db8:0a0b:12f0:0000:0000:0000:0001 becomes 0x20010db80a0b12f00000000000000001 as octet string output.
