SPR_ACCOUNT_NEW source context

Table: SPR_ACCOUNT_NEW source context
Attribute name Trigger Description
Account-id SUBSCRIBER_PRE_PROVISIONING This attribute provides the account id of the account.
Note:
Only the first account of the subscription is considered for this attribute. If device has multiple subscriptions, purchased by multiple accounts, then this attribute is filled by the first fetched account of the subscription.
Account-Name SUBSCRIBER_PRE_PROVISIONING This attribute provides the account name of the account.
Subscriber-Custom-Attribute SUBSCRIBER_PRE_PROVISIONING This attribute provides the user custom data.
