Generate EDR for step-up bucket

The ME flag Generate EDR for step-up of Charging Step bucket is introduced to control the behavior of EDR generation when step-up happens for Charging step bucket. The flag is of type ENUM and has the following values:

  • ENABLE_FOR_STEP_UP_ON_RESERVATION - The EDR is generated only when the Step-Up On Commit flag is set to False.
  • DISABLE - The EDR generation is disabled.

Default value is Disable.

If... Then...
The flag is set to ENABLE_FOR_STEP_UP_ON_RESERVATION EDR needs to be generated for every step-up of Charging step bucket.
The flag is set to DISABLE No EDR is generated when step-up happens for Charging step bucket.
