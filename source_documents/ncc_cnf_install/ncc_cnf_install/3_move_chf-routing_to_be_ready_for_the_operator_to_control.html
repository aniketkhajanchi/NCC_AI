Edit the chf-routing Helm chart

Because the chf-routing Helm chart is already deployed, you need to alter it so that the NCC operator can begin controlling and making changes to it.

Run these commands on the namespace where NCC is already installed. Adjust the namespace value as needed.

namespace=ig1
helm_release=$(helm list -o json -n ${namespace} | jq -r '.[] | select(.name | contains("ig-chf-rou")) | {name: .name} | .name')
newreleasename=ig-chf-routing
 
kubectl -n $namespace delete secret -l name=${helm_release},owner=helm
 
RESOURCES=$(kubectl -n $namespace get all,cm,vs -o jsonpath="{ range .items[?(@.metadata.annotations.meta\.helm\.sh/release-name=='$helm_release')] }{.kind}/{.metadata.name}{'\n'}{end}")
for RESOURCE in ${RESOURCES}; do
    kubectl -n $namespace annotate --overwrite ${RESOURCE} "meta.helm.sh/release-name"=$newreleasename
done
