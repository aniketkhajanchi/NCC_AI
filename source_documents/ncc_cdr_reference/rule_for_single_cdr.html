Rule for single CDR

The action CALL_RESULT.Enable-Single-CDR is available in the following triggers to configure single CDR for IMS, SMS, and MMS calls:

  • IMS_CALL_ESTABLISHMENT (for IMS)

  • SMS_CALL_ESTABLISHMENT (for SMS)

  • MMS_CALL_ESTABLISHMENT (for MMS)
