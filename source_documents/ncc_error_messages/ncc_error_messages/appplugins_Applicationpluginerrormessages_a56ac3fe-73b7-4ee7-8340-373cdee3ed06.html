Application Plugin Error Messages

This table describes the application plugin error messages that the NCC may generate.

Table: Application plugin error messages
Number Message HTTP Error Code Description
PLUGIN_00001 Request sequence is null. 500 The request sequence cannot be null.
PLUGIN_00002 Response sequence is null. 500 The response sequence cannot be null.
PLUGIN_00003 Invalid plugin name length in request sequence 500 Plugin name {0} in request sequence is not between {1} and {2} characters in length.
PLUGIN_00004 Invalid plugin parameter length in request sequence 500 Plugin parameter {0} in request sequence is not between {1} and {2} characters in length.
PLUGIN_00005 Plugin name/parameter combination in request sequence not unique. 500 Plugin name/parameter pair of {0}, {1} has been duplicated in the request sequence.
PLUGIN_00006 Invalid plugin name length in response sequence 500 Plugin name {0} in response sequence is not between {1} and {2} characters in length.
PLUGIN_00007 Invalid plugin parameter length in response sequence 500 Plugin parameter {0} in response sequence is not between {1} and {2} characters in length.
PLUGIN_00008 Plugin name/parameter combination in response sequence not unique. 500 Plugin name/parameter pair of {0}, {1} has been duplicated in the response sequence.
PLUGIN_00009 Conflicting plugin configurations. 500 Plugin configurations cannot exist for the Integrated system type and also exist for the I/O or Processor system types.
PLUGIN_00010 Duplicate plugin configurations. 500 Only one plugin configuration can exist per system type.
PLUGIN_00011 Plugin does not exist. 500 No plugin named {0} exists in the system.
PLUGIN_00012 Invalid plugin in request sequence. 500 Plugin {0} does not support requests so it cannot be in a request sequence.
PLUGIN_00013 Invalid plugin in response sequence. 500 Plugin {0} does not support responses so it cannot be in a response sequence.
PLUGIN_00014 Unsupported system type. 500 Plugin {0} does not support the system type {1}.
PLUGIN_00015 System type not set. 500 The system type is not set.
PLUGIN_00016 Request and response sequences are both empty. 500 The request and response sequences cannot both be empty.
