Prerequisites

Before installing CCTF on Nokia Container Services (NCS​), ensure that the following prerequisites, as well as Installation prerequisites are met:

  • NCS is available. For NCS cluster requirements, see Cluster.
  • The CCTF chart is available in a local chart repository.
  • All images needed by the CCTF chart are available in a local Docker registry:
    • cctf/cctf:<version>
    • cctf/cctf-jenkins:<version>
    • cctf/cctf-tools:<version>
    • jenkins/jnlp-slave:<version>
    • dduportal/bats:<version>
    • cbur/cbura:<version>
    • cmdb/admin:<version>
    • cmdb/mariadb:<version>
    • test execution environment image (file name: bash_robotsps_docker-<version>.tar.gz, Docker image name: bash:<version>)
  • (Optional) CITM chart is available in a local chart repository if CITM is chosen as the ingress controller.
  • (Optional) All images needed by CITM are available in a local Docker registry if CITM is chosen as the ingress controller.
