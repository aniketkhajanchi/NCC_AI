Administration

Note:
The CCTF administration functionalities can be performed only by an administrator.

The following table presents the privileges of the three role types in CCTF (that is, Viewer, Operator and Administrator).

Role management

If a Keylocak server is configured to one CCTF server, the role management is handled through Keycloak. If not, the role management is handled through the CCTF UI, in the User Management view, which can be entered by clicking the icon.

To change a user role through the CCTF UI

  1. In the User Management view, click the edit icon on the row of the user whose role you want to change.

    An update user role view open.

  2. Change the user's role to a desired one.
  3. Click SAVE.

Operation logs

The daily operations on CCTF are recorded with the information of user name, time, operation type and results. An administrator can check and manage the operation logs in the Operation Log view, which can be entered by clicking the icon. The following figure presents an example of the Operation Log view.

Figure: Operation Log view

Operation logs can be deleted per record (by clicking the delete icon on the log row) or in a batch (by clicking DELETE ALL LOGS).

Note:
The deleted operation logs cannot be recovered.
