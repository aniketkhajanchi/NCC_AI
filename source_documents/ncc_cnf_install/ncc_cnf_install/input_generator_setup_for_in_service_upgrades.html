Configuring the Input Generator for an ISU

The Input Generator spreadsheet (Inputs.xlsx) must be configured for an ISU. For more information on the spreadsheet, see Preparing Helm chart input values.

  1. Set the Deployment Type value to in-service upgrade in the Global section of the Common worksheet.
  2. Set the MariaDB Simplex value to false in the Global section of the Common worksheet, for geographic redundancy.
  3. Set the Database Persisted Version value to the iBuild number of the deployed release in the SPS section of the Common worksheet.
  4. Set the MariaDB Replication Enabled value to true in the CKEY section of the namespace worksheet that you are upgrading, for geographic redundancy.

What to do next

TLS parameters (Cipher/TLS Mode) for Aerospike in Inputs.xlsx cannot be changed during an ISU. It is also recommended not to modify the configurations or extend the resources of the charts during an upgrade. For each namespace, use the Helm upgrade commands to upgrade the NCC charts.

For more information on Helm chart dependencies, see Helm charts deployment order and dependencies.
