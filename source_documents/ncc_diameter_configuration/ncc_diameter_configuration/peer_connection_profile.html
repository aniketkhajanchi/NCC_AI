Profiles

To configure a profile, login to SM GUI and navigate to Connectivity → ☰ → DIAMETER PEERS → Diameter Peers. Select PROFILES toggle, and click + to select one of the following profiles to be created:
  • Peer Connection Profile
  • Ingress Throttling Profile
  • Egress Throttling Profile
Peer Connection Profile

A peer connection profile specifies the values to be used for:

  • Timers

  • Capabilities exchange

One of the peer connection profiles is set as NCC system default profile and is used when peer-level values are not provided.

The applications for both authorization and accounting applications are as follows:

To configure a Peer Connection Profile, enter the parameter details as shown in the following table:
Table: Peer Connection Profile parameters

Parameter

Description

Optional or Mandatory
Profile Name Specify the profile name. Mandatory
Description Specify a description of the peer connection profile. Optional
Profile Details
Origin Host Specify the name or IP address of the origin host. The host name must not contain the “/” character. Optional
Origin Realm Specify the name or IP address of the origin realm. Optional
Origin State ID Specify the origin state ID to be used for establishing a connection with the peer. Optional
Vendor ID Specify the vendor ID for the profile. Optional
Product Name Specify the product name that NCC advertises to external clients, such as an AF server. Optional
Authorization and Accounting
Authorization Applications

List one or more authorization application types to reach a specified host or realm. Choose one or more application types from the drop down.

By default, all applications are valid for a route. Leave the field blank to enable all applications.

Click the +AUTHORIZATION IDS button and enter the Vendor Id and Application.

 Mandatory
Accounting Applications

List one or more accounting application types to reach a specified host or realm. Choose one or more application types from the drop down.

By default, all applications are valid for a route. Leave the field blank to enable all applications. 

Click the +ACCOUNT IDS button and enter the Vendor Id and Application.

 Mandatory
Custom AVP

Click Custom AVP button to add one or more AVP type and name associations.

Specify a Attribute Name for the AVP entity.

Select the Type of the AVP from the drop-down list and provide the applicable Value.

 Optional
