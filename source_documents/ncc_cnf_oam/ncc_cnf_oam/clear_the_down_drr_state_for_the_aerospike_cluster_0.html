Clear the DOWN_DRR state for the Aerospike cluster

Run this command to clear the DOWN_DRR state.

# NOTE: This will clear the DOWN_DRR state in case if it was set for the system. 
kubectl -it exec <admincli> -c <container> -n <namespace> -- /opt/tpa/sbin/db-roster.py clear
