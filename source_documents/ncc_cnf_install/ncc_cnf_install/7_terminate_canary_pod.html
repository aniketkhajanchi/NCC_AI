Terminating the canary pod

Edit the spreadsheet again to disable traffic routing to the canary pod.

  1. Open the Inputs${CHF_SOLUTION_VERSION}.xlsx file.
  2. Set these values in the namespace worksheet in the CHF Traffic Routing section:
      • Type: NONE
      • Percentage New: empty (blank)
      • All Canary Active values: false
  3. Save and close the spreadsheet.
  4. Repeat the steps to regenerate the custom resource.
  5. Run this command to apply the changes. 
    kubectl -n ncc-operators apply -f output/chargingfunction-ig1.yaml
