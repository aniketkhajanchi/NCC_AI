Upload parameters file from local

PATH: api/lab/editparameter/?labID={lab ID}

METHOD: PUT

Example:

PUT
https://xxx.xxx.xxx.xxx/cctf/api/lab/editparameter/?labID=1
request body:
Content-Disposition: form-data; name="myfile"; filename="parameters.json"
Content-Type: application/json

response body:
{
  "res": 0,
  "resMsg": "Upload parameter file successfully",
  "obj": {
    "name": "parameters.json",
    "size": "0.001MB"
  }
}

Delete test package by lab ID

PATH: api/project/instance/{lab ID}/

METHOD: DELETE

Example:

DELETE
https://xxx.xxx.xxx.xxx/cctf/api/project/instance/3/
response body:
{
  "res": 0,
  "resMsg": "Deleted Successfully",
  "obj": ""
}
