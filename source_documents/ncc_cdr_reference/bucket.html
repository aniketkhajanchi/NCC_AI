Bucket

EDR tags

EDR Tag Description Example
bucketName Indicates the name of the Bucket. Buc1
bucketUnitType Indicates the unit type for the entity. Byte
bucketKindOfUnit Indicates the kind of unit for the entity. VOLUME
reUnitTypeCreationTime Specifies the time at which the entity was created. 03/12/2018 14:15:21
initialValue Specifies the Bucket initial value. 2500
maxCarryOverValueOption Indicates whether carryover is of type absolute or percentage. ABSOLUTEVALUE
maxCarryOverValue

The maximum balance that the carryover bucket can hold at the next billing cycle.

 1200
bucketTransferLimit The maximum carryOverValue. 100
renewalPeriodMode Renewal period mode of the bucket. ONE_PERIOD
consumptionPriority Consumption priority of the bucket, carryover first or main bucket first.

Possible values: Regular_bucket, carry_over

 CARRY_OVER_BUCKET
thresholdProfileGroupIdList The threshold group definition names. [Netflix_Threshold.Threshold]
bucketCreationTime Specifies the time at which the entity was created. 02/07/2018 12:37:53
bucketLastUpdateTime Specifies the time at which entity was last modified. 02/07/2018 12:37:53
bucketBalanceBefore Bucket initial balance.

For maximum value, refer to Max value for Bucket and Counters.

 1048576000
maxRollOverValue The maximum balance that the rollover bucket can hold at the re-subscription. 2147483648
