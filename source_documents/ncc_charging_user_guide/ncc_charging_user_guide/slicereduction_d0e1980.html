Configuration

Follow the given step to configure slice reduction:

  1. Calculate the delta (D) between current usage (committed quantity + reserved quantity) and upcoming threshold values.

  2. Apply the Slice Allocation Factor (SAF) value on the above delta (D) value using the Slice Allocation Factor parameter defined in the quota management (slicing profile) rule.

  3. Compare the above calculated value with the Requested Service Units (RSU) and identify the smaller value as SMIN.

  4. Compare the MIN Slice Size or SAS value with SMIN and grant the one with larger value.
