CDRs for NCHF AMF (N41) and NEF (N44) session

CDR tags

The CDR tags for NCHF AMF (N41) and NEF (N44) calls are same as that of NCHF (N40) call (see CDRs for NCHF (N40) session). However, the tags mentioned in the following table are either different or have different meanings from the corresponding N40 tags.

Table: Additional CDR tags for NCHF AMF (N41) calls
NCC record tag Description/source parameter in HTTP request Example
RegistrationChargingInformation   RecordProperty
N2ConnectionChargingInformation    
LocationChargingInformation    
Table: Additional CDR tags for NCHF NEF (N44) calls
NCC record tag Description/source parameter in HTTP request Example
exposureFunctionAPIInformation    
Note:
Only the first value is dumped for externalGroupIdentifier as specified in 3GPP 32298 Rel 16 specs.
