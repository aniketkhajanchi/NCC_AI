CG Kafka Distributor Node cannot send a message to Apache Kafka brokers

If CG Kafka Distributor Node cannot send a message to Apache Kafka brokers, it shows an alert message and forwards the record to the Log distributor stream. To restore the connectivity to Apache Kafka brokers, make sure that

  • the value of node parameter BootstrapServers is correct
  • the necessary topics have been created in Apache Kafka or the Apache Kafka brokers configuration parameter auto.create.topic.enable is set to true.
