Installing CCTF on NCS

This procedure describes how to install CCTF on Nokia Container Services (NCS).

Execute the following commands from the Kubernetes NCS control node as root.

  1. Create an override file cctf-overrides.yaml with the given content: 
    cat <<EOF > cctf-overrides.yaml
threeButtonsAPI:
  enabled: true
storageSize: "2Gi"
controller:
  config:
    proxy-body-size: 100m
EOF
  2. Get the CCTF Helm chart (part of the NCC delivery).
  3. Install CCTF with name cctf in namespace cctf, and with the values defined in the override file (in step 1): 
    helm install cctf-<version>.tgz --name cctf --namespace cctf -f cctf-overrides.yaml
  4. Verify the installation:

    1. Check that the CCTF pod exists in the cctf namespace: 
      kubectl get pods -n cctf

      Example output:

      NAME                            READY   STATUS    RESTARTS   AGE
cctf-cctf-5f8df9b7bd-f2b6g      2/2     Running   0          82s
cctf-cmdbmariadb-mariadb-0      1/1     Running   0          82s
cctf-jenkins-6c7dc784b9-4w6mp   1/1     Running   0          82s
    2. Check that the CCTF service exists in the cctf namespace: 
      kubectl get svc -n cctf

      Example output:

      NAME                     TYPE        CLUSTER-IP       EXTERNAL-IP   PORT(S)          AGE
cctf-cctf                ClusterIP   10.254.247.193   <none>        80/TCP,443/TCP   94s
cctf-cmdbmariadb-mysql   ClusterIP   10.254.78.154    <none>        3306/TCP         94s
cctf-jenkins             ClusterIP   10.254.90.123    <none>        8080/TCP         94s
cctf-jenkins-agent       ClusterIP   10.254.129.60    <none>        50000/TCP        94s
