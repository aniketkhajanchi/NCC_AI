Donor-Secondary-Balance SUBSCRIBER_POST_TRANSFER

SUBSCRIBER_FAILED_PROVISIONING

This attribute returns the secondary balance of the donor account.

The following attributes are made available for GIFTING source context, from the Donor Account:
  • Donor-Secondary-Balance.Bucket-Name: The name of the bucket used while rating a call.
  • Donor-Secondary-Balance.Value: This attribute provides the value.
  • Donor-Secondary-Balance.Initial-Value: This attribute provides the bucket initial value.
  • Donor-Secondary-Balance.End-Date: This attribute provides the end date of a periodic lifecycle attached with an account.
An index shall enable to identify the specific Secondary-Balance value. Example: Donor-Secondary-Balance.Bucket-Name (x), Donor-Secondary-Balance.Value (x).

If Donor-Secondary-Balance is not present, the return value is NULL.
