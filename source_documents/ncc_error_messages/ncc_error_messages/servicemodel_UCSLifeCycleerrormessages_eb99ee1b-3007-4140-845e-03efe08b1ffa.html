Ucs Lifecycle Error Messages

This table describes the UCS LifeCycle error messages that the NCC may generate.

Table: UCS LifeCycle error messages
Number Message HTTP Error Code Description
LIFECYCLE_00001 Life Cycle does not exist. 500 Lifecycle does not exist. Enter an existing one.
LIFECYCLE_00002 Life Cycle type is invalid. 500 Only Period type life cycle is allowed.
LIFECYCLE_00003 Life Cycle entity type mismatch. 500 Life Cycle entity type is mismatch.
LIFECYCLE_00004 Life Cycle name missing. 500 Life Cycle name is missing.
UCSLIFECYCLE_00001 UCS Lifecycle object is missing. 500 UCS Lifecycle object must be specified.
UCSLIFECYCLE_00002 UCS Lifecycle object Id is missing. 500 Id is a mandatory field and must be specified.
UCSLIFECYCLE_00003 UCS Lifecycle object Id already exists. 500 The object with ID "{0}" already exists in DB.
UCSLIFECYCLE_00004 UCS Lifecycle object Rules are missing. 500 Rules are mandatory and must be specified in UCS Lifecycle object "{0}".
UCSLIFECYCLE_00005 Multiple rules are defined in UCS LifeCycle Object. 500 Multiple rules cannot be added for UCS Lifecycle object "{0}".
UCSLIFECYCLE_00006 No actions are defined in UCS LifeCycle Object rule. 500 Actions are mandatory for UCS Lifecycle object "{0}" rule.
UCSLIFECYCLE_00007 UCS LifeCycle Object does not exist in database. 500 UCS Lifecycle object "{0}" not found in database.
