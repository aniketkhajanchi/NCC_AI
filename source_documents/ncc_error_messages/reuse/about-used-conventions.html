Conventions used

Appearance Description
emphasis Text that is emphasized.
graphical user interface text Text that is displayed in a graphical user interface.
system input Text that the user types as input to a system.
system output Text that a system displays or prints.
variable A value or command-line parameter that the user provides.
[ ] Text or a value that is optional.

{value1│value2}

{variable1│variable2}

 A choice of values or variables from which one value or variable is used.
Note:
Be cautious while copying and pasting commands from any PDF. Due to the in-built hyphenation and line break features of various PDF readers, copying may introduce a line break or removal of hyphens if they appear at the end of a command line. Verify before running the commands as-is.
