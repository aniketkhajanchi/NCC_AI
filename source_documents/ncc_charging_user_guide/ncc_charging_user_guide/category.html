Logic Categories (Category Order)

The category order entity is used by the operators to define the categories and their order of consumption. This entity is used to select the charging services.

A default category order is created at the time of NCC installation with the given values:

  • Name = DefaultCategoryOrder

  • CategoryList = DefaultCategory

Configuration

Log into the new GUI, navigate to Charging → ☰ → Charging Management → Logic categories.

Click EDIT CATEGORY against the default category consumption order entity to modify the entity details as described in the following table:

Note:
The operators can only edit the default category consumption order entity configuration. You cannot create or delete a category consumption order entity.
