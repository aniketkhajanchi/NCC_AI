Uploading additional files to an existing test package

A new UI entry is added to enable uploading additional files to an existing CCTF test package. The folder to hold the uploaded files is pre-defined as uploaded_file under the top folder of a test package, where also the parameters.yaml file is saved. Only one file is allowed to be updated each time when uploading files to an existing test package. But multiple files can be uploaded through multiple tries for one CCTF test package.

  1. Click the three-dot button besides the name of the test package to be updated, for example:
  2. Select Upload File from the list.
