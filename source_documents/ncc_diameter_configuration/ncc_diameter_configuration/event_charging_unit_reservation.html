Event Charging with Unit Reservation (ECUR) support in the NCC Gy interface

NCC does not differentiate between Session Charging with Unit Reservation (SCUR) or ECUR, hence ECUR is treated like the SCUR, where no CCR-Update request is exchanged between network and Charging application.
