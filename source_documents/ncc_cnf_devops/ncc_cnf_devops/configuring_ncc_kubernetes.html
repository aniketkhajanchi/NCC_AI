Configuring NCC Kubernetes

NCC Kubernetes needs to be set up for test suite execution with Airflow by opening outbound notification port to Maestro VM.

Example:

Run the following command in the Kubernetes platform of NCC, using Maestro VM IP 10.1.1.4:

cat <<EOF | kubectl apply -f –
apiVersion: networking.istio.io/v1beta1
kind: ServiceEntry
metadata:
name: sps-me-notfi-out
namespace: sps-btel-me
spec:
addresses:
- 10.1.1.4/32
hosts:
- any
location: MESH_EXTERNAL
ports:
- name: http-for-notif-server
number: 11200
protocol: TCP
resolution: NONE
EOF
