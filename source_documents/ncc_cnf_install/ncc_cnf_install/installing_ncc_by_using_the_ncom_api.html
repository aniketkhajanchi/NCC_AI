Installing NCC by using the NCOM API

Follow this procedure to install NCC by using the NCOM API and scripts that are created for installation.

  1. Generate a network service (NS) package by using the procedure found in Generating Network Service Package in input generator.
  2. Prepare NCS by using the procedure found in Preparing NCS.
  3. Generate a chartsOverrideInputs.json file by using the tool yaml-To-Json.py.
    For more information on the tool, see Tool location.
  4. Edit the deployNS.conf file located in the input_generator/output/SITE_TYPE/examples/NCOM directory.
    The values are explained inside the file itself.
  5. Run this command to upload NSD followed by creation and deployment of NS. 
    ./deployNS.sh -f deployNS.conf
