Export and import

Overview of export and import using User and Role Management GUI

The User and Role Management GUI allows you to import and export most of the resources in the User and Role Management server database.

Note:
The attributes containing secret or private information are masked in the export file.

The files created during a startup export can also be used to import from the admin user interface, which enables to export from one realm and import to another realm. Alternatively, export from one server and import to another.

Note:
The User and Role Management GUI exports or imports allows just one realm per file.

The User and Role Management GUI import operation allows you to overwrite resources depending on the selection.

The User and Role Management GUI export operation allows you to export clients, groups, and roles. If there is a great number of any specific assets in the realm, then this operation may take more time than usual to complete the entire process. During the transition of the export operation, the User and Role Management server may not be responsive to user requests.
