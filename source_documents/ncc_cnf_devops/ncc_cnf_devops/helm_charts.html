Helm charts

The following Helm charts are used to setup the framework based on Kubernetes:

  • The NGINX chart (citm-ingress) is used to route the requests from end users.
  • The base Framework chart (CCTF) brings up the back-end and front-end of the framework, which work cooperatively to load the test package, select test cases, trigger test execution and feedback test results.

    The base Framework chart includes the following sub-charts:

    • The database of the framework is implemented by the DB chart (cmdb), and only the back-end of the framework can access the database.
    • The Jenkins chart (jenkins) sets up a Jenkins server, which is used to schedule the test execution, bring up the test container to run tests, create the test log, and feed test results to the framework back-end.
