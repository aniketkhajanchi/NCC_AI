Redelivery stream scaling

The CGF (DR) SFTP Kafka Failure Collector Node and CG Kafka Distributor Node can be scaled by increasing the number of node instances, but it must be done with caution to ensure that the other nodes in the stream can handle with the load. There can be several Kafka redelivery streams working in parallel.
