Generating Network Service Package in input generator

This procedure provides instructions on how to create the Network Service Package/Descriptor (NSD) in the input generator.

Navigate to the /output/<elementName>/examples/NCOM directory of the input generator and run the generate-network-service-descriptor.sh script.
This script creates the NetworkServiceDescriptor(rendered).zip named NS package. For more information on the script, refer to Tool location.
