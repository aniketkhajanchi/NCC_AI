SS7 to NCC: SOAP/REST integration

Notification messages come from NCC to APIGW through FEP SOAP. For other features, NCC-SS7 sends requests to NCC through FEP-SOAP. For successful calls, you must complete these basic configurations for the USSDGW and APIGW services. You can edit the configurations via the OSP UI or the SS7 Manager GUI. Details are found in the NCC SS7 User Guide.

APIGW Service Configuration object

Provide the primary or secondary URL and their respective port to trigger NCCNCC. Provide the USSDGW Sep Group name that is used to link to other USSDGW SEPs. The Sep Group can be created in the sepgroup object of the pfmconfig service.

APIGW USSD response code object

This object is used to relay the HTTP code towards NCC. For Balance Query and other features (except Notification), select the request name from the drop down list. Configure messages code for each HTTP response code. The message code is sent to USSDGW with HTTP code.

USSDGW Global Configuration object

Select IMSI or MSISDN from the Global Config object based on which HLR is picked to send messages towards HLR.

Note:
The Search Prefix value is used for notifications.

USSDGW Communication Entity object

Create all of the entities with which the USSDGW service interacts. To communicate with the APIGW service, create an entity for APIGW and provide its Sep Group name. All of the entries are preconfigured for communication purposes.

USSDGW Service Configuration object

Configure the service-side Global Title SCP address to communicate with any external node.

USSDGW Retailer object

Configure the Retailer options for Balance Query, Bundle Balance Query, and language update type calls. For NCC, the retailer name value should be Default.

USSDGW Language object

An instance of this object is preconfigured for messages sent towards the HLR.

USSDGW Message Component object

Create the message text for Balance Query, Bundle Balance query, and language update. The object is preconfigured with default text.

USSDGW Message Composer object

Create messages by using the fixed components from the Message Component object and variable components from this object to format a complete message. This object is preconfigured to compose the message with a combination of fixed and variable text.

USSDGW Application object

For direct access calls, configure the USSD access code for Balance Query, Bundle Balance query, and language update.

USSDGW Menu Tree object

For menu-based calls or composite menu calls, create menu objects for specific USSD codes.
