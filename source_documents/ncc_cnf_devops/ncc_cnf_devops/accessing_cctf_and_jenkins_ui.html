Accessing CCTF UI and Jenkins UI

This procedure describes how to enable the access to the CCTF UI and Jenkins UI.

  1. Depending on the OpenShift configuration, map the istio-ingressgateway node IP (or if loadbalancer is installed, the IP of loadbalancer) to the host names defined in cctf-openshift-overrides.yaml.
  2. Update the Domain Name System (DNS) or hosts.

    For example, on a Windows desktop, add the mappings in the C:\Windows\System32\drivers\etc\hosts file, for example:

    10.x.x.x espoo-cctf.com

    10.x.x.x espoo-jenkins.com

  3. Access the UIs using the correct target port. The target port for 443 is 31390.

    By using the examples given here, you can:

    • access the CCTF UI through https://espoo-cctf.com:31390/
    • access the Jenkins UI through https://espoo-jenkins.com:31390/jenkins
