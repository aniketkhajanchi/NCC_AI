Allowance Limits (Bucket)

Allowance limits (buckets) are entities defining an amount and unit of consumption. The allowance limit (bucket) is defined with a name to be referenced by the tariff and charging service. It is a reusable entity and same bucket definition can be used by multiple charging logic (charging service).

Configuration

Log into the new GUI, navigate to Charging → ≡ → COMPONENT LIBRARY → Allowance Limits, and click to add a new allowance limit (bucket).

Description

resetBucketAndItsThresholds Responsible for fetching the bucket instance using the id. Resets the current value of bucket instances to match the provided resetValue (BigDecimal). updateBucketInstance
