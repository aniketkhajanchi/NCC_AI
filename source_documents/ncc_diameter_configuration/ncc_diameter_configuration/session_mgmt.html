Session management

NCC provides the ability to manage (terminate, audit) Sy, HTTP N28, Ro, Gy, N40 sessions. See the following sections in the Charging User Guide to understand these parameters and their usage in more detail:

  • Proactive clearing of stale Diameter/Nchf_ConvergedCharging sessions
  • Sy/N28 session management

To configure the session management parameters, in SM GUI click Connectivity → ☰ → Session Management.

Session management parameters are available separately for 4G and 5G. Select 4G or 5G, or both using the GUI buttons in the Session Management window.

For 4G, the session management parameters are available under the following:
  • CHF SY
  • Diameter 4G
For 5G, the session management parameters are available under the following:
  • NCh Spending Limit Control
  • Nchf 5G N40
