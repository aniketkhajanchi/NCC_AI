Uninstall the operators

Note:
The NCC operator now controls the chf-routing Helm chart, which is key for the process. It should be left running, even after the canary test is completed. Only uninstall the operator if you are removing the entire NCC application.

Run these commands to uninstall the NCC operator:

kubectl -n ncc-operators delete chf chargingfunction-ig1
 
helm -n ncc-operators delete chf-operator
 
helm -n ncc-operators delete app-api
 
helm -n ncc-operators delete ccpm-resource 
 
# To cleanup CSF operator CRDs from cluster
helm delete -n ncc-operators app-api-crds-only
 
kubectl delete ns ncc-operators
