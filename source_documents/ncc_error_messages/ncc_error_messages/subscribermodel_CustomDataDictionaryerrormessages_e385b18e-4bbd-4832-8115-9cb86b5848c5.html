Custom Data Dictionary Error Messages

This table describes the Custom Data Dictionary error messages that the NCC may generate.

Table: Custom Data Dictionary error messages
Number Message HTTP Error Code Description
CUSTOMDATADICT_00001 Invalid custom data dictionary attribute(s). 500 Custom data dictionary attribute(s) "{0}" are invalid.
CUSTOMDATADICT_00002 Custom data dictionary attribute is a mandatory field. 500 The custom data dictionary attribute "{0}" field must be provided "{1}".
CUSTOMDATADICT_00003 Custom data dictionary attribute defined multiple times. 500 The attribute "{0}" already exists in the custom data dictionary.
CUSTOMDATADICT_00004 Required custom data attribute not present. 500 The custom data dictionary defines attribute "{0}" is required for "{1}".
CUSTOMDATADICT_00005 Custom data not defined in custom data attribute dictionary. 500 The custom data attribute "{0}" does not exist in context "{1}".
CUSTOMDATADICT_00006 Custom data dictionary not loaded. 500 The custom data dictionary could not be loaded.
CUSTOMDATADICT_00007 Custom data dictionary attribute does not exist. 500 The attribute "{0}" does not exist in the custom data dictionary.
CUSTOMDATADICT_00008 Custom data dictionary attribute is in use by active Rule or Subscriber objects. 500 The attribute "{0}" is immutable because it is used by either an active rule or a subscriber entity.
CUSTOMDATADICT_00009 Read-only custom data cannot be updated. 500 The custom data attribute "{0}" is defined as "read-only" so that the custom data cannot be updated.
