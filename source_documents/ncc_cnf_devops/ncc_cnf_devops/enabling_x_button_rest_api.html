Enabling X-button REST API

By default, X-button APIs are disabled. To enable them, set threeButtonsAPI.enabled=true during the CCTF is installation, for example:.

helm install incubator/cctf -n cctf --set threeButtonsAPI.enabled=true --namespace cctf

If the switch has not been turned on during the deployment, it can be turned on also with the helm upgrade command.

Note:
There is no authentication before these REST APIs are called, which might cause security risks.
