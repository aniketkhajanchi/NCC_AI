Creating a telemetry admin user

Perform these steps to add a telemetry admin user.

  1. Log into the User and Role Management GUI as the administrator.
  2. Select the btel-auth realm.
  3. Click on Manage > Users.
  4. Click on Add user.
  5. Fill out these fields:
    • User Name:
    • Email:
    • First Name:
    • Last Name:
    • User Enabled: ON
    • Email Verified: ON
    • Groups: GrafanaAdmin
  6. Click Save.
  7. Select the newly created telemetry admin user.
  8. Click on the Credentials tab.
  9. Create a default password for the user.
  10. Click Reset Password.
  11. Click on the Role Mappings tab.
  12. Add these Realm Roles:
    • kibanauser
    • belkadmin
    • offline_access
    • uma_authorization
    • read_alarm_role
    • update_alarm_role
    • ack_alarm_role
    • clear_alarm_role
  13. Select account from the Client Roles drop-down.
  14. Add these roles:
    • manage-account
    • manage-account-links
    • view-profile
