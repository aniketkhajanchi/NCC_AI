Mapping details for CHF records in predefined ASN.1 schema

This chapter describes the detailed rules for mapping supported CHFRecord tags as per 3GPP specification 32.298 v15.11.0. NCC only supports the tags described in this chapter (the 32.298 v15.11.0 specification contains more tags) and 3GPP specification 32.298 v16.7.0. For more information on the CHFRecord schema, refer to section 5.2.5 of the 3GPP specification 32.298 v15.11.0.

NCC generic records are mapped to CHFRecord V15 or V16 for a NCHF (N40) call. For more information on the mapping rules, see Mapping rules for CHF V15 records and Mapping rules for CFH V16 records

Rating related information is not supported in the CHFRecord specification and therefore not available in CHFRecord.
