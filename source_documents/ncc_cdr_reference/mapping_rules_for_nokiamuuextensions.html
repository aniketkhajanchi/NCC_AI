Mapping rules for NokiaMUUExtensions

Table: Mapping rules for NokiaMUUExtensions
Output field name Input field name Conversion rule

recordEventType

<parent>. recordEventType

Map string to integer as follows:

PS -> 46

SMS -> 25

MMS -> 143

NCHF_CHARGING -> 500

IMS ->168

recordEventResult

<parent>.recordEventResult

Directly map the first value found for recordEventResult.

Validate that the value is between 0 and 1023. If it is not, raise a warning and do not map.

chargingService

<parent>.groupInfo*.subscriptionInfo*.chargingServiceInfo*.chargingServiceName

Make a list of all chargingServiceName values.

Direct mapping:

chargingServiceName -> UTF8String

callCost

<parent>.groupInfo*.subscriptionInfo*.accountInfo.callCost

<parent>.groupInfo*.subscriptionInfo*. bucketInfo.callCost

Directly map the first value found for callCost.

callCost can be under accountInfo or bucketInfo.

If the value has more than 64 characters, take the first 64 characters.

tariffPlan

-listOfMscc.mscc*.groupInfo*.

subscriptionInfo*.accountInfo.tariffID

-listOfMscc.mscc*.

groupInfo*.subscriptionInfo*. bucketInfo.tariffID

Directly map the first value found for tariffID to tariffPlan.

tariffPlan can be under accountInfo or bucketInfo.

If the value has more than 64 characters, take the first 64 characters.
