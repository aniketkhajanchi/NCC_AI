Retrieving NCC artifacts from the Support Portal

This procedure provides instruction on how to retrieve NCC installation artifacts such as Helm charts, Docker images, and CSF configuration examples from Support Portal.

  1. Log in to Support Portal via this link: https://download-na.support.nokia.com/cgi-bin/Download.pl.
  2. Select Nokia Converged Charging from the Product drop-down list.
  3. Select the version from the Hierarchy drop-down list and wait for the list to populate (for example, NCC_22_9_CNF).
  4. Select the Input Generator package.
    The package name is in this format: input-generator-chart-<release version>.tgz.
  5. Select all of the images you wish to download.
  6. Click the Next button.
  7. Follow the directions given on-screen to download the files.
  8. Put each downloaded NCC installation package into its own folder.
