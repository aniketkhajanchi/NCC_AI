Configuring SS7 values for the ss7sim chart

This step must be performed to configure the ss7sim chart.

Edit the values.yaml file and add these lines. 
global:  
  registry: <image registry>
elementType: "ig"
where:
  • <image registry> is the repository where the ss7sim image is located.
  • elementType: is either "ig" or "me" based on the deployment type.
