Max value for Bucket and Counters

The maximum supported value for Buckets and Counters is 1125899906842624 (in base units) with DB precision = 2. With increase in DB precision, this value decreases. For example, if DB precision is set to 5, the maximum supported value becomes 92233720368547.

Note:
The maximum value as described in this section is applicable for the following tags in all CDRs and EDRs:
  • counterValueBefore
  • counterValueBefore
  • bucketBalanceBefore
  • bucketBalanceAfter
  • accountBalanceBefore
  • accountBalanceAfter
  • accoutBalanceBR
  • accountBalanceBeforeRenew
  • accountBalanceAfterRenew
  • currentStep
