Segregation of data - Charging and Common

Only the entity types listed in this section are segregated per service provider for Charging and Common. That is, entities created in a service provider are available (and can be associated/used by other entities) within that service provider. These are not available for other service providers.

Entities with the same name can exist across service providers. However, the database identifier of such entities are prefixed with the service provider name.

An MNO user can view, create, edit, publish data for segregated entities for any MVNO or MNO.
