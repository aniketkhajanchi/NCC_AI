Variable preference order

It is expected to define only the required variables and keep the remaining variable values as None. But if the user fills all the values, the following order should be honored:

  1. helm-chart-dir and cnf-namespace over sdc configuration files for either IG or Distributed setup, that is, if helm-chart-dir, cnf-namespace, and sdc files are all present, then DAG will be generated for SPS-CNF considering values for helm-chart-dir and cnf-namespace.
  2. igsdc over mesdc or smsdc: If either of medsc or smsdc is configured, and also igsdc is configured, then DAG will be generated for IG SPS-VNF.
  3. mesdc and smsdc over igsdc: If both mesdc and smsdc are configured along with igsdc, then DAG will be generated for Distributed SPS-VNF.
  4. vnf files will only be considered for DAG generation if sdc files are configured.
  5. To receive e-mail, all e-mail variables need to be configured.
