Notification Service EDRs

The Notification Service application generates Event Data Records (EDRs) only for notification events. Provisioning events do not generate EDRs. Also, EDRs are not generated for exceptions.

To enable EDR generation in NS, the following application preference is used.

Table: NS EDR application preference

Application Preference

Description

Option
Notification EDR Logging Enabled Enable or disable EDR generation for NS application.

boolean (true, false)

Default: false
