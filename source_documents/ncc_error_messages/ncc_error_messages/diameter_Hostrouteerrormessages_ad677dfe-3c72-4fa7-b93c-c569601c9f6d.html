Host Route Error Messages

This table describes the Host route error messages that the NCC may generate.

Table: Host route error messages
Number Message HTTP Error Code Description
HOSTROUTE_00001 Invalid FQDN. 500 The value {0} is not a valid host route FQDN (fully qualified domain name).
HOSTROUTE_00002 Invalid realm. 500 The value {0} is not a valid host route realm.
HOSTROUTE_00003 Invalid route type. 500 The value {0} is not valid.
HOSTROUTE_00004 Invalid priority. 500 The value {0} is not in the valid priority range of {1} to {2}.
HOSTROUTE_00005 Invalid application. 500 The value {0} is not a valid Diameter application.
HOSTROUTE_00006 Description is too long. 500 The description you have provided is {0} characters. The Description field must be between 0 to 255 characters.
HOSTROUTE_00007 Duplicate route configuration. 500 The host route configuration for application {0}, peer {1} and realm {2} conflicts with another host route configuration for that peer.
HOSTROUTE_00008 Cannot modify the type field. 500 The type field of a host route configuration cannot be modified.
HOSTROUTE_00009 Cannot delete a system-defined route configuration. 500 A system-defined host route configuration cannot be deleted.
HOSTROUTE_00010 Cannot delete active route configuration. 500 A host route configuration cannot be deleted when it is active.
