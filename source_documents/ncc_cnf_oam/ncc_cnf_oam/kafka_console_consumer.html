kafka-console-consumer

Description

Use this tool to read data from Kafka topics and display it to standard output. The tool is located in the /opt/tpa/bin directory.

Syntax

kafka-console-consumer.py [-h] --broker-list <BROKER_LIST> --topic <TOPIC> [--from-beginning]
Table: Options
Option Description
-h or --help Displays usage/help information.
--broker-list <BROKER_LIST> Kafka broker addresses separated by comma.
--topic <TOPIC> Kafka topic name.
--from-beginning If the consumer does not already have an established offset to consume from, start with the earliest message present in the log rather than the latest message.

Examples

1. Consume messages from a topic.
/opt/tpa/bin/kafka-console-consumer.py --broker-list ig-ckaf-kafka-headless.ig1.svc.cluster.local:9092 --topic Test_topic_1

Output:
None: b'First Message without key'
None: b'Second Message without key'

2. Consume messages from a topic since the beginning of the log. Note that messages which are deleted during Kafka's cleanup activity are not available.
/opt/tpa/bin/kafka-console-consumer.py --broker-list ig-ckaf-kafka-headless.ig1.svc.cluster.local:9092 --topic Test_topic_1 --from-beginning

Output:
None: b'First Message without key'
None: b'Second Message without key'

3. Sample output when messages in the topic are given by key.
/opt/tpa/bin/kafka-console-consumer.py --broker-list ig-ckaf-kafka-headless.ig1.svc.cluster.local:9092 --topic Test_topic_1

Output:
b'test_key1': b"Another sample message with key 'test_key1'"
b'test_key1': b"Sample message with key 'test_key1'"
