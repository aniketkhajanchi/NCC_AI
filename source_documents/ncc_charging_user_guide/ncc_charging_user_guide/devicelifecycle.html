Device lifecycle configuration

The system supports only the ENTITY lifecycle for devices. As a lifecycle entity, a device must be associated with an ENTITY lifecycle. If you do not associate an ENTITY lifecycle with a device, then the system associates this device with DefaultDeviceLifeCycle.

You can configure various ENTITY lifecycles for devices and choose any of them to associate with a device. You can associate a device only with one ENTITY lifecycle.

Note:
Once a lifecycle is created, it cannot be deleted.
