Mapping details for EDRs in predefined ASN.1 schema

This section describes the detailed rules for mapping the following Lifecycle and eCommerce EDRs from the generic CDR schema to the predefined ASN.1 schema.

Lifecycle EDRs:

  • renewAccountRecord
  • renewSubscriptionRecord
  • resetAccountRecord
  • resetSubscriptionRecord

eCommerce EDRs:

  • checkBalanceRecord
  • debitAmountRecord
  • getUCBalanceRecord

For more information on the predefined schema, see Predefined ASN.1 schema.
