Mapping rules for PDU address in CHF records (V15)

Table: Mapping rules for PDU address CHF records (V15)
Output field name Input field name Conversion rule for NCHFChargingRecord (5G)
pDUIPv4Address

<parent>.pDUIPv4Address

Direct mapping.

Convert pDUIPv4Address into binary IP address with pDUIPv4Address as parent (see IP address conversion to binary IP address). If the input is not a valid IP address, raise a warning and do not map.

pDUIPv6AddresswithPrefix

<parent>.pDUIPv6AddresswithPrefix

Direct mapping.

Convert pDUIPv6Address into binary IP address with pDUIPv4Address as parent (see IP address conversion to binary IP address). If the input is not a valid IP address, raise a warning and do not map.
iPV4dynamicAddressFlag <parent>.iPV4dynamicAddressFlag

Direct mapping:

iPV4dynamicAddressFlag -> iPV4dynamicAddressFlag

Verify that the flags are valid Boolean values (0 or 1). Raise a warning and do not map if they are not.
iPV6dynamicPrefixFlag <parent>.pDUAddress.iPV6dynamicPrefixFlag

Direct mapping:

iPV6dynamicPrefixFlag -> iPV6dynamicPrefixFlag

Verify that the flags are valid Boolean values (0 or 1). Raise a warning and do not map if they are not.
