EDRs for Sy/N28 sessions

This section provides information about the EDR tags for Sy and N28 sessions. Generation of Sy/N28 EDR is disabled by default. It can be enabled using the following parameters:

  • Application preference OCS Sy CDR Generation (see section Managed Application Preferences in Global Configuration Guide).
  • Attribute Generate SNR CDR (see section Outbound Configuration in Diameter/HTTP Configuration Guide).
