Configuration at the NCC client in User and Role Management

At the User and Role Management server, select the realm that is configured for NCC (SPS-AUTH) and then from the Clients menu and navigate to the client configured for the NCC (spssm).

On the Settings tab in the Client menu, add an entry as /ss7manager/* in Valid Redirect URIs.

Click on the Save button.
