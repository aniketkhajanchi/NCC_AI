dayOfWeek

This attribute indicates the day of week.

A value of 7 indicates EXACT. This indicates that the day of week is the same as the day of week of the subscription purchase date.

 Yes Yes Yes
disableTTC

This tag signifies if TTC calculation should occur. If this flag is set to true, then TTC calculation is not done and VT is aligned to the expiration date and time of the subscription.

Possible values: true, false

 Yes Yes Yes
accountID Account ID with which subscription is funded. Yes Yes Yes
hourOfDay

This attribute indicates the hour of day.

A value of 24 indicates EXACT. This indicates that the time in end date and time is same as subscription start time.

A value of 25 indicates start of new day. The end date and time are aligned to 00:00:00 of next day, as per the lifecycle periodicity.

 Yes yes yes

The deviceId tag indicates that the subscription is attached to a device. The groupID indicates that subscription is attached to a group.
