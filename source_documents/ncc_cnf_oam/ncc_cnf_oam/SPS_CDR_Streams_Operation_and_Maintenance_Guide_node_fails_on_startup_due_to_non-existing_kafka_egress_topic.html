CG Kafka Collector Node fails on startup due to non-existing Kafka Egress topic

Make sure that the value set in node parameter EgressTopicName is valid. It should also be verified that the topic was created in the Apache Kafka broker.

Run this command to check the topic existence in Kafka server host:

kafka-topics --list --bootstrap-server localhost:9092

If the topic is not listed, create it with an adequate number of partitions:

kafka-topics --create --bootstrap-server localhost:9092 --replication-factor <replication-factor> --partitions <partition-count> --topic <topic-name>

For more information, see the Apache Kafka documentation.
