Log distributor stream

Log Distributor stream handles records that Kafka output stream has failed to distribute. When Kafka Distributor Node fails to distribute a record, it saves this record (encoded in ASN.1 format) to the database. The node then sends a message to a given log queue to notify Log Distributor Node that it needs to pick up the failed record and store it to a remote location. Therefore it is important that node parameter LogQueueName is set to the same value in CG Kafka Distributor Node and CG Log Distributor Node.
