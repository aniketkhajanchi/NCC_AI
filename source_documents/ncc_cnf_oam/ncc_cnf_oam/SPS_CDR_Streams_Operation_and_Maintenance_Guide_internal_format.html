Internal format

The distribution rule configuration is stored within the database under key {Config}:DistributionRules_v2 in JSON format.

For backward compatibility and to enable in-service upgrade, the old version of the configuration is kept in the database with key {Config}:DistributionRules. The old configuration will be eventually cleaned up once the system has been fully upgraded.

There is no need to examine this value unless you are instructed to do so by support for debugging purposes or if the lookup table data gets corrupted and there is a need to recover the current configuration.
