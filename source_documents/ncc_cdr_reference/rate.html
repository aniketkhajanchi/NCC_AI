Rate

EDR tags

Tag Description Example
rateId This parameter indicates the ID of the rate associated with the entity. Rate_money
connectionCost This parameter specifies the connection cost defined in this Rate entity. 10
rateUnitType* This parameter specifies the unit type of rate and granularity level. Second
rateKindOfUnit* This parameter specifies the kind of unit of rateUnitType. TIME
ratereUnitTypeCreationTime* This parameter is the time at which the associated re unit type is created was created. 06/07/2021 12:00:33
reUnitTypeId - -
rateCreationTime This parameter is the time at which the Rate was created. 06/07/2021 12:00:33
rateLastUpdateTime This parameter is the time at which the Rate was last modified. 06/07/2021 12:00:33
taxIncluded This parameter indicates whether applied tax is included to the fee/cost or not.

If true, tax is included to the cost/fee.

If false, tax is applied over and above cost/fee.

Default Value: false

 true
