Creating product

In CCTF, products are used to organize test packages within one CCTF. Each test package belongs to one and only one product, but one product can have multiple test packages. The following procedure describes how to create a new product.

Note:
If the desired product already exists, there is no need to create a new one.

  1. In CCTF UI, click ADD PRODUCT under the product list.
    An Add Product pop-up window opens.
  2. Enter a new product name and click ADD.
