Sample CDRs for SMS/MMS

This section provides sample CDRs for mobile originated multi-recipient SMS, and MMS calls.

MO MR-MMS success

If MO multi-recipient MMS call is successful, then separate CDRs are generated for each recipient, and a single CDR with details of all the recipients and consolidated charging information is also generated. For example: If numberOfRecipients = 10, then 10 CDRs are generated along with a single consolidated CDR.

The following CDR is generated separately after processing the call for the first recipient address.
