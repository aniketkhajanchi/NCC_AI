Maestro VM test suite execution in Airflow

Test suites (DAGs) can be run with Maestro VM in an NCC CNF deployment using Airflow.

There are two options to run test suites through Airflow UI:

  • DAG execution - Disabled by default. Can be enabled by the user by configuring Airflow variables. Available for all Airflow/Maestro users.
  • Three-button UI - Simplified specific UI. Supposed to be used by customers in their labs, or having a team assisting them for automation.
