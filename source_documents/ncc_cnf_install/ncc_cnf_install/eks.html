EKS

EKS is an object input where you can configure many of the EKS cluster attributes.

Attribute name Default value Mandatory Type Description
cluster-name null yes string Name of the AWS EKS cluster.
subnets null yes list of subnets IDs as a JSON array. At least 2 subnet identifiers in different availability zones.
vpc-id null yes string vpc identifier where EKS has to be created.
security-groups null yes string default security groups used for EKS.
service-ipv4-cidr 172.20.0.0/16 no string cidr for Kubernetes service to use.
tags null no map of string key value pairs to tag nodes and cluster.
