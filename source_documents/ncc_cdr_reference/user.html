User

EDR tags

EDR Tag Description Exmple
userID It contains the userId. If the application preference AutoGen SPR Ids is set to true then this tag contains the value of userName. UserTest_1
userDbId The user identifier stored in database. The userDbId tag appears only if the application preference AutoGen SPR Ids is set to true.

7893ae9f-5a56-4221-8f92-0bfa7a3a68dc
userName The user name stored in the database. UserTest_1
lastName Last name of the user. Sd
firstName First name of the user. Mak
phoneNumbers Phone numbers of the user. 9999999999
emails Email of the user. skhflkseh@maktest.com
facebookId Facebook ID of the user. facebookIdUser
userCreationTime Creation time of entity. 03/08/2018 12:26:40
lastUpdateTime Last updated time of entity. 03/08/2018 12:26:40
entityLifeCycleName Entity lifecycle name. entl1
entityLifeCycleState Entity lifecycle state. Active
homeSite Indicates the ID of the home site on which the entity is homed. 6
partition Indicates the partition in which the entity is created. 6
localSite Indicates the ID of the local site from which the entity is created. 6
subscriberTracingId

If subscriber tracing is enabled for a Device, then this tag is added and it contains the Device ID.

 D1
subscriberTracing Indicates whether subscriber tracing is enabled. true
