Tools used

This table shows which tools are used for disaster recovery procedures.

Tools Description
xdr-status Displays XDR metrics and status for the local and remote sites.
db-stats Displays Aerospike database statistics.
db-backup Supports database backups.
db-restore Restores the complete NCC database from a backup copy that was made previously with the db-backup tool
TPA_repairKeyGen Generates a new key that is in turn used to generate club IDs
disaster-recovery-helper.py This script helps in performing database related operations required during recovery
db-prepare-rewind Prepair sites for xdr-rewind
