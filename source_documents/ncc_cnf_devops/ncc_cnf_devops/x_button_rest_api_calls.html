X-button REST API calls

In the following examples, <test suite name> is defined in cctf_buttons.json by testSuiteName, if it exists. Otherwise <test suite name> is defined by testType.

Trigger test job with test suite name:

Path: api/experimental/dags/<test suite name>/dag_runs
Method: POST
For example:
http_server_ip + /api/experimental/dags/ChargingNewFeaturesSuite/dag_runs
http_server_ip + /api/experimental/dags/PerformanceChargingSuite/dag_runs
http_server_ip + /api/experimental/dags/ChargingRegressionSuite/dag_runs


Get the job list with test suite name:

Path: api/experimental/dags/<test suite name>/dag_runs
Method: GET
For example:
http_server_ip + /api/experimental/dags/ChargingNewFeaturesSuite/dag_runs
http_server_ip + /api/experimental/dags/PerformanceChargingSuite/dag_runs
http_server_ip + /api/experimental/dags/ChargingRegressionSuite/dag_runs


Monitor job status with test suite name and job start time:

Path : api/experimental/dags/<test suite name>/dag_runs/<job time>/tasks/<test suite name>stats
Method: GET
For example:
http_server_ip + /api/experimental/dags/ChargingNewFeaturesSuite/dag_runs/2020-08-26T07:43:27+00:00/tasks/CheckChargingNewFeaturesSuitestats
http_server_ip + /api/experimental/dags/PerformanceChargingSuite/dag_runs/2020-08-26T07:43:27+00:00/tasks/CheckPerformanceChargingSuitestats
http_server_ip + /api/experimental/dags/ChargingRegressionSuite/dag_runs/2020-08-26T07:43:27+00:00/tasks/CheckChargingRegressionSuitestats


Get the report file with test suite name and job time(latest only for GUI):

Path : /admin/automationreports/<test suite name>/<job time or latest>/report.html
Method: GET
For example:
http_server_ip + /admin/automationreports/ChargingNewFeaturesSuite/latest/report.html
http_server_ip + /admin/automationreports/PerformanceChargingSuite/latest/report.html
http_server_ip + /admin/automationreports/ChargingRegressionSuite/latest/report.html


Get the log file with test suite name and job time(latest only for GUI):

Path : /admin/automationreports/<test suite name>/<job time or latest>/log.html
Method: GET
For example:
http_server_ip + /admin/automationreports/ChargingNewFeaturesSuite/latest/log.html
http_server_ip + /admin/automationreports/PerformanceChargingSuite/latest/log.html
http_server_ip + /admin/automationreports/ChargingRegressionSuite/latest/log.html
