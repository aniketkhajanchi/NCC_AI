Application preferences

Set the value of the following application preferences as follows to enable this feature:

  • Set the value of N28 Pending Policy Counter to Time_Aligned_With_TTC
  • Set the value of N28 Pending Policy Counter to Time_Aligned_With_TTC

Refer to section Charging application preferences for details about these application preferences.

Note:
For this feature to work, the feature described in section Sy and N28 keep-alive notifications and message throttling must be enabled.
