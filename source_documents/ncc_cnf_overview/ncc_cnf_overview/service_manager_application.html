Service Manager

This module is the gateway for receiving various provisioning requests over REST interface from the operator's IT system and performs the provisioning of service, subscriber, and system-related data and publishing them to various ME subsystems. It interacts with the Provisioning component to perform data publishing in ME subsystems. For more information about the Service Manager subsystem, see Service Manager subsystem.
