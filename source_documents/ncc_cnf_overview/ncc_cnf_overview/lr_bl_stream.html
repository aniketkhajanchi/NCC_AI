Late rating business logic stream

Late rating business logic stream maps the late rating records into Diameter AVP format. The stream consists of one standalone node, Late Rating Mapper Node. The node reads the records stored into the database by Late rating input stream and maps them according to late rating mapping rules. It then writes the mapped records into the database for Late Rating Diameter Distributor Node to read. No distribution rules are added. Late Rating Mapper Node also validates the length of some of the input fields, causing invalid fields to be excluded from the mapped records.

Scaling

Late Rating Mapper Node can be scaled by adjusting the number of parallel connections to database, defined with node parameter DatabaseClientPool, as well as the concurrency used when reading session records, defined with node parameter Concurrency. There can also be several Late rating business logic streams working in parallel. The different Late Rating Mapper Node instances do not interfere with each other when reading from the same database.
