EDR aggregation use cases

Following are some use cases for EDR aggregation:

  • Receiving aggLastRecord= true with no gaps on the basis of aggSequenceNumber with result = Success for all requests within a single session generate a final output with result = Success.
  • Any missing request in the session leads to generation of a final output with result = Fail after getting the SessionTimeout passes.
  • Any failure for a single request is considered as failed output.

All the mentioned cases generate partial EDR outputs dumped into the partial directory and Kafka topic.

Aggregated EDRs are populated under edr/final file directory and edr-final Kafka topic. Partial EDRs are populated under edr/partial file directory and edr-partial Kafka topic.
