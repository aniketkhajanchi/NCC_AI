Zoning And Roaming Mapping Error Messages

This table describes the zoning and roaming mapping error messages that the NCC may generate.

Table: Zoning and Roaming Mapping error messages
Number Message HTTP Error Code Description
ZONINGANDROAMINGMAPPING_00001 MCCMNC and Address are missing. 500 The MCCMNC and Address, one of them must not be empty.
ZONINGANDROAMINGMAPPING_00002 Invalid MCCMNC. 500 The MCCMNC is invalid.
ZONINGANDROAMINGMAPPING_00003 The MCCMNC is duplicated. 500 The MCCMNC is already used by the record with id "{0}".
ZONINGANDROAMINGMAPPING_00004 The address is duplicated. 500 The address is already used by the record with id "{0}".
ZONINGANDROAMINGMAPPING_00005 Invalid Address. 500 The value of Address is not valid. It must be in the format of "type{0}value", and the type must be among "{1}" with uppercase letters.
ZONINGANDROAMINGMAPPING_00006 Invalid Ipv6 Address. 500 The value of Ipv6 "{0}" is not valid.
ZONINGANDROAMINGMAPPING_00007 Invalid Ipv4 Address. 500 The value of Ipv4 "{0}" is not valid.
ZONINGANDROAMINGMAPPING_00008 NetworkType and zones are missing. 500 The networkType and zones, one of them must not be empty.
ZONINGANDROAMINGMAPPING_00009 One of zones is empty. 500 Zone cannot be empty.
ZONINGANDROAMINGMAPPING_00010 Country is a blank string. 500 Country can be empty but cannot be a blank string.
