Restore dsclocal

On the recovering site, restore the dsclocal namespace of the database by issuing the following command on the control node of the recovering site.

kubectl -it exec <admincli> -c <container> -n <namespace> -- /opt/tpa/bin/db-restore -d /opt/tpa/data/db-backup/db-backup -n dsclocal -f
Where:
  • <admincli> is the name of the admincli pod.
  • <container> is the name of the container.
  • <namespace> is the name of the namespace.
