Installing Helm charts manually

This procedure provides instructions on how to install Helm charts manually. This option can also be used to re-install a particular component or exclude a component.

Run this command to generate a list of example commands. 
cd input_generator/output/SITE_TYPE/examples
 
./display-helm-commands.sh
Note:
This script goes through the generated install-charts.sh script and echoes the commands rather than executes them.
