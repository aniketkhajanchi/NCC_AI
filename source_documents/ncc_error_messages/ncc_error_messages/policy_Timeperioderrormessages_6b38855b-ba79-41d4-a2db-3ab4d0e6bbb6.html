Time Period Error Messages

This table describes the time period error messages that the NCC may generate.

Table: Time period error messages
Number Message HTTP Error Code Description
TP_00001 Name of the time period is not unique. 500 Name of time period "{0}" must be unique. Modify the name and try saving again.
TP_00002 Name is a mandatory field. 500 The mandatory Name field provides a unique name for this object and must be 1 to 80 characters.
TP_00003 Name exceeds the maximum length of 80 characters. 500 The Name field must be 1 to 80 characters.
TP_00004 Invalid Start Time of Day. 500 The Start Time-of-Day must be between 0 and 86399999.
TP_00005 Invalid End Time of Day. 500 The End Time-of-Day must be between 0 and 86399999.
TP_00006 Invalid Day of Week. 500 The Day-of-Week must be one of SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY or SATURDAY.
TP_00007 Invalid start day and time. 500 Start day and start time must be either both specified or both not specified for a time-of-week.
TP_00008 Invalid end day and time. 500 End day and end time must be either both specified or both not specified for a time-of-week.
TP_00009 Missing time period component. 500 Time period must contain at least one time-of-day, day-of-week or time-of-week.
TP_00010 Invalid combination of time period components. 500 Time period must not contain time-of-day or day-of-week if any time-of-week are specified.
TP_00011 Rule engine reload failed. 500 Failed to reload rule engine after updating time period "{0}".
