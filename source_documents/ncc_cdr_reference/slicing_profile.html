Slicing Profile

EDR tags

Tag Description Example
slicingProfileName This parameter is the unique name of a Slicing profile. default_profile
sliceSize This parameter indicates the static size of slice in Bytes. 512
quotaHoldingTime This parameter has value in seconds, which is used to control the time the session remains in the idle state in the network. 5
validityTime This parameter has value in seconds which is used to control the time the quota remains active in the network and the effective allocation of resources. 10
slicingCreationTime This parameter is the time at which the Slicing profile was created. 02/07/2018 12:31:26
slicingLastUpdateTime This parameter is the Time at which the Slicing profile was last modified. -
ignoreReservedQuota

This parameter determines whether the reservation quota must be considered in slice reduction logic or not. The default value of this parameter is No.

Possible values: No, Counter
