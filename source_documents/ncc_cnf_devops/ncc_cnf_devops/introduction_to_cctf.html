Introduction

CSF Common Test Framework (CCTF) is a cloud-native application built on top of Kubernetes, which allows users to port their testing environment and perform test activities to qualify deliverable products as part of the continuous integration (CI) and to verify the products in customer premises as part of the continuous delivery (CD).

The purpose of this document is to provide guidelines to CCTF end users and help them to understand how to:

  • deploy a CCTF instance
  • run test cases on CCTF
  • check the execution results and log files.
