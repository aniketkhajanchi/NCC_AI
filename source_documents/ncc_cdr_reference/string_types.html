String types

Input values can be of two string types: UTF8String and OctetString. Output fields can have the following string types: UTF8String, GraphicString, IA5String and OctetString.

The mapping from an OctetString to an OctetString output field is a direct copy of the bytes. The mapping from a UTF8String to an output field of type UTF8String, GraphicString or IA5String is also a direct copy of the content. These three types have slightly different character sets, but in this context they are all printable ASCII characters.

The mapping from UTF8String to an OctetString output field is a conversion of the ASCII characters of UTF8String to octet format.
