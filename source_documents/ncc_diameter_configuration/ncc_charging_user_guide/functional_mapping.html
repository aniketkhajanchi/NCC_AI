The new launcher can be accessed using one of the following ways:

  • access the URL as in https://<SM_IP>:<SM_port>/ncc/

    OR

  • in the classic launcher, click the Go to New Launcher icon.

Table: Banner components in the new GUI
Icon marker Description
1 Indicates the NCC application, for example, Charging, Platform, Notification.
2 Click to expand the list of entities available in the application.

Configuring objects

In the new launcher, navigate to Charging/Platform/Notification/Connectivity and select components from the hamburger menu (☰). It displays the existing components created. You can also perform the following actions:

  • Click at the top-right corner of the page to create a new object.
  • Select an existing object and click to edit that object details.
  • After committing one or more configuration changes to any charging component, save the change and publish by clicking . This opens a Publish Version form that allows you to select one or more available ME as the target for change.
