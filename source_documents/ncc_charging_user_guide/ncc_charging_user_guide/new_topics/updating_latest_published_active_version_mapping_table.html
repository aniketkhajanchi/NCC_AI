Updating latest published active version (mapping table)

Backdating of mapping table
The latest published active version of a mapping table can be fixed (updated or edited) and published. The fixed version of the latest published active version is called a Patch Fix or Backdated Fix. The effective date of the Patch Fix is same as that of the latest published version, regardless of when the Patch Fix is published.
