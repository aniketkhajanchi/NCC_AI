Custom CDR tags

The CDR tags described in this section are additional tags meant for specific customers. Following configuration is provided in application preferences to enable these tags.
Note:
This document presents standard CDR/EDR structures generated by NCC. However, location of some tags may change inside a given group structure for different scenarios.
Table: Application preference for enabling custom CDR tags

Parameter

Type

Description
CDR Custom flag boolean

If this flag is set to true, then custom CDR tags are supported.

Valid values: true, false

Default value: false

Notice:
This parameter is meant for specific customers and is not validated for generic use. Enabling this parameter in a production environment may impact CDR generation. Consult with Nokia Support before enabling this parameter.
