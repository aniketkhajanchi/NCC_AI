Is-Receiver

SUBSCRIBER_PRE_TRANSFER

SUBSCRIBER_POST_TRANSFER

 This attribute returns a boolean value. If the current agent type is from the receiver, then this attribute returns true. Else, it returns false.

 This attribute returns a boolean value. If the current agent type is from the sender, then this attribute returns true. Else, it returns false.
Is-Retry

SUBSCRIBER_PRE_TRANSFER

SUBSCRIBER_POST_TRANSFER

 If the activity type of the transfer request is retry, then this attribute returns true. Else, it returns fasle. This attribute is used for handling rollback requests.
Is-Rollback

SUBSCRIBER_PRE_TRANSFER

SUBSCRIBER_POST_TRANSFER

 If the activity type of the transfer request is rollback, then this attribute returns true. Else, it returns fasle. This attribute is used for handling rollback requests.
Transfer-Unit

SUBSCRIBER_PRE_TRANSFER

SUBSCRIBER_POST_TRANSFER

 This attribute identifies the type of unit sent in the TransferBucket API. It can have the following values:
  • TIME
  • VOLUME
  • MONEY
  • UNIT
Units (basic or derived) must be defined in NCC.
Transfer-Fee

SUBSCRIBER_PRE_TRANSFER

 This attribute defines the transfer fee.
Transfer-Amount

SUBSCRIBER_PRE_TRANSFER

 This attribute defines the transfer amount.
Time-Start-Of-Today-TZ

SUBSCRIBER_PRE_TRANSFER

 This attribute defines the start of the day time.
Current-Date-Time-TZ

SUBSCRIBER_PRE_TRANSFER

 This attribute defines the current date time.
