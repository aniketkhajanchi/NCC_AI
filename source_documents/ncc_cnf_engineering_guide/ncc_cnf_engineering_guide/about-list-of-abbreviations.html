Table: List of abbreviations

Term

Full form
Call Model The combination of diameter message types (Gy, Sy etc.) and diameter/API TPS. The subscriber/session count is also part of the call model.
TPS Transactions per second. A transaction is defined as a diameter or API request-response.
Database The Aerospike database which holds all objects such as subscribers and sessions.
