Plan change

When a plan is changed, the invoke method in the EDR is change_bundle. Data for both the old bundle and new bundle are also generated.

EDR tags

Tag Description Example

This tag signifies the type of bundle change.

This tag signifies whether the bucket is a carryover bucket.
